### 背景コンテキスト（SoEフレームワーク）

私はSoE（Service of Empowerment / Service or Exploitation）という研究フレームワークを構築しています。中核的な主張は以下の通りです：

1. **複式簿記メタファー**: 個人の人生を企業会計に見立て、資産（能力・権利）と負債（トラウマ・社会的障壁）を貸借対照表（BS）で、日々の変化を損益計算書（PL）で、行動エネルギーをキャッシュフロー（CF）で追跡する
2. **トバシ（Tobashi）検出**: 社会（親会社）が本来負担すべき教育・配慮コストを未払いにし、その結果生じた個人（子会社）の不全を「自己責任」としてBSに付け替える構造的不正を検出する
3. **ACEsとの連結**: 小児期逆境体験（ACEs）は個人の「長期借入金」であり、教育段階でのInput不足が「資産の減損未処理」を引き起こし、そのまま労働市場に投入される
4. **Input Constitutional AI**: 支援記録の作成段階で権利侵害を検出・防止するAIシステムの提案

**この文脈で、子どもの権利条約は「社会（親会社）が子ども（子会社）に対して負っている債務の根拠法」として位置づけられます。**

### 調査項目（すべて日本語で出力）

#### 1. 子どもの権利条約（CRC）の基本構造

- 採択年（1989年）・発効年（1990年）・日本の批准年（1994年）
- 条約の4つの一般原則（General Principles）:
  - 第2条（差別の禁止）
  - 第3条（子どもの最善の利益）
  - 第6条（生命・生存・発達の権利）
  - 第12条（意見表明権）
- 全54条のうち、SoEフレームワークに特に関連する条文を特定・解説

#### 2. SoEに直接関連するCRC条文の詳細分析

以下の各条文について、**条文の原文（日本語訳）・趣旨・SoEフレームワークとの対応関係**を示してください：

| 条文 | テーマ | SoEとの対応（想定） |
|------|--------|---------------------|
| 第2条 | 差別の禁止 | 障害児への教育機会の不平等 = トバシの前提条件 |
| 第3条 | 最善の利益 | 「子どもの最善の利益」が就労支援段階まで連続しているか |
| 第6条 | 生存・発達 | 発達権 = BS上の資産形成への投資義務 |
| 第12条 | 意見表明権 | Input Constitutional AIの法的根拠 |
| 第19条 | 虐待からの保護 | ACEsの発生防止 = 負債の未然抑止 |
| 第23条 | 障害児の権利 | 障害児への特別な配慮 = 追加的Input投資の義務 |
| 第24条 | 健康 | 心身の健康 = CF（キャッシュフロー）の基盤 |
| 第27条 | 生活水準 | 適切な生活水準 = 最低限のBS資産保障 |
| 第28条 | 教育への権利 | 教育機会 = 資産形成プロセスそのもの |

#### 3. 日本における国内法への反映

CRCの理念がどの国内法に具体化されているかを一覧化してください：

- **児童福祉法**（1947年制定、2016年大改正） — 特に2016年改正で「子どもの権利条約の精神にのっとり」が明記された点
- **児童虐待防止法**（2000年） — 第19条との関係
- **教育基本法**（2006年改正） — 第28条・29条との関係
- **障害者基本法** — 第23条との関係
- **子ども基本法**（2023年施行） — CRC全体の国内実装
- **こども家庭庁設置法**（2023年） — 制度的実装
- **子どもの貧困対策法**（2013年） — 第27条との関係
- **いじめ防止対策推進法**（2013年）
- **少年法** — 司法領域での子どもの権利
- その他関連法令

各法律について、**CRCのどの条文を具体化したものか**の対応関係を明示してください。

#### 4. 国連子どもの権利委員会による日本への勧告

- 日本政府報告書に対する総括所見（Concluding Observations）の要点
  - 第1回（1998年）、第2回（2004年）、第3回（2010年）、第4-5回（2019年）
- 特に以下の点について委員会が指摘した懸念事項：
  - 競争的教育環境と子どもへの過度な圧力
  - 障害児のインクルーシブ教育の不足
  - 子どもの意見表明権の不十分な実装
  - 虐待対応体制の不備
  - 代替養育（社会的養護）の課題

#### 5. CRC × ACEs × 教育機会毀損の連鎖分析

以下のストーリーラインに沿って、CRC違反がどのように連鎖するかを分析してください：

```
[Stage 1] 小児期にACEs発生
  → CRC第19条（虐待からの保護）違反
  → CRC第6条（発達権）の阻害
  → BS上の負債（長期借入金）蓄積

[Stage 2] 学齢期に教育機会が毀損
  → CRC第28条（教育への権利）の形骸化
  → CRC第29条（教育の目的＝人格形成）の未達成
  → CRC第23条（障害児の特別配慮）の不履行
  → BS上の資産（教育的資本）が減損未処理

[Stage 3] 社会に送り出される（就労移行段階）
  → 卒業証書は存在するが中身が伴わない（含み損の連結飛ばし）
  → 自己責任論によるトバシの完成
  → SoE判定：Service or Exploitation?
```
