# Applied Behavior Analysis (ABA) — エビデンスソース一覧
> 作成日: 2026-03-10
> 分類: 介入手法 / 認知・非認知エビデンス

## 認知的エビデンスの概要
タスク分析（Task Analysis）、体系的指導（Systematic Instruction）、刺激制御（Stimulus Control）がABAの認知的フレームワークです。ABA利用者は複雑な職業行動を細分化されたステップに段階的に習得し、各段階における刺激条件と応答パターンを認知的に内在化させます。この学習プロセスを通じて、環境シグナルへの認知的反応性が強化され、職業場面での自動的判断能力が醸成されます。

## 非認知的エビデンスの概要
目標設定（Goal Setting）、選択意思（Choice-Making）、自己管理（Self-Management）、適応行動（Adaptive Behaviors）がABAの非認知的コンポーネントです。ABAは報酬構造を利用して利用者の内発的動機づけを段階的に構築し、外部強化から内部管理への移行を促進します。この過程において、利用者の自律性、自己効力感、職業行動の安定性が向上し、長期的な就労継続を支援します。

## SoEフレームワークとの関連
ABAは SoE会計的隠喩における「行動資本の累積」（Behavioral Capital Accumulation）モデルとして機能します。個別の行動技能の習得を「資産計上」と見なし、強化スケジュールの段階的変更を「資本効率化」として理解します。エンパワーメント評価の観点では、ABAプログラムの実装者と利用者の両者が行動変化の測定と評価に直接参加し、データドリブンな改善サイクルを通じて個人的および組織的な学習と成長を実現することで、真の参加的エンパワーメントが達成されます。
