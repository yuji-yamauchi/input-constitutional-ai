# MENTA 活動記録

## ディレクトリ構造

```
MENTA/
├── mentees/           # メンティー別フォルダ
│   └── {名前}/        # 個別プロジェクト・セッション記録
├── templates/         # 再利用可能なテンプレート
├── assessments/       # メンター評価・振り返り
├── session_logs/      # 汎用セッションログ
└── README.md
```

## 機密性について

このディレクトリはGitHubリポジトリ外に配置。
メンティーの個人情報・事業情報を含むため、公開リポジトリには含めない。
