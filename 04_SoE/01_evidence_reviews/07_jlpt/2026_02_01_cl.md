# 日本語能力試験（JLPT）N認定制度の包括的調査報告

**2010年改定で導入された5段階N認定システムは、項目応答理論（IRT）に基づく尺度得点により公平な能力測定を実現し、年間147万人が受験する世界最大規模の日本語能力評価基盤となった。** JLPTの最大の特徴は「読む・聞く」の受容技能に特化した測定であり、「話す・書く」は評価対象外である。この設計上の制約は、2025年12月から成績書類に追加表示されるCEFR対応においても明示され、N1合格者でもC1相当は「受容活動のみ」と注記される。Can-do statementsによる自己評価研究は、試験得点では捕捉できない能力の可視化を試みており、その「〜ができる」形式の記述構造は、Senのケイパビリティアプローチにおける「機能（functionings）」の記述様式と構造的類似性を持つ。

---

## 1. JLPT制度の全体像

### 歴史的経緯と2010年改定の意義

JLPTは**1984年**に国際交流基金と日本国際教育協会（現・日本国際教育支援協会）の共催で開始され、当初は年1回・4段階（1級〜4級）の試験であった。**2010年7月の大規模改定**で現行の5段階システム（N1〜N5）に移行し、旧2級と旧3級の間に**N3を新設**して「橋渡しレベル」を創出した。「N」は**Nihongo**と**New**の頭文字を表す。

2010年改定の核心的変更点は3つある。第一に、素点から**尺度得点（Scaled Score）**への移行により、試験回間の難易度変動を調整し得点の比較可能性を確保した。第二に、「言語知識」だけでなく「**課題遂行のための言語コミュニケーション能力**」を測定対象とする理念の明確化。第三に、総合得点と各区分基準点の**両方を満たす合否判定方式**への変更で、バランスの取れた能力を要求するようになった。

### 実施規模と国際的位置づけ

**2024年の受験者数は年間約147万人**（応募者172万人）で過去最高を記録し、**96カ国・地域**で実施されている。2017年に応募者数100万人を初突破して以降、コロナ禍の2020年を除き増加傾向が続く。実施主体は、海外を**国際交流基金**、日本国内を**日本国際教育支援協会**が担い、試験の企画・開発は両組織の共催で行われる。

他の日本語能力評価との関係では、**BJT**（ビジネス日本語能力テスト）がビジネス場面に特化し、**NAT-TEST**がJLPTと同等構成で年6回実施、**JFT-Basic**が特定技能1号向けの基礎テストとして機能している。法務省は留学ビザ申請に際しJLPT以外の9試験も認定しているが、JLPTが国際的認知度と在留資格要件としての採用において最も広範な地位を占める。

## 3. 得点算出・合否判定システムの仕組み

### 尺度得点とIRTの適用原理

JLPTの得点は「正答数×配点」の素点ではなく、**項目応答理論（IRT: Item Response Theory）**に基づいて算出される**尺度得点**である。IRTでは各問題に「困難度」「識別力」などのパラメータが設定され、受験者の**解答パターン全体**から能力値（θ）を確率論的に推定する。同じ正答数でも難しい問題に正答した受験者はより高い尺度得点を得る設計となっている。

**等化（Equating）**は異なる試験回間の難易度差を調整する手法で、共通項目（アンカー項目）を各回に配置し、その結果を基準として得点尺度を統一する。公式ガイドブックの例では、同じ正答数10問でも「難しい回」での受験者は「やさしい回」より**5点高い尺度得点**を得る。この仕組みにより、**いつ受験しても同じ能力であれば同じ尺度得点**が算出され、受験者は自身の能力の伸びを客観的に確認できる。

### 合否判定の二重条件と基準点の意義

合格には**①総合得点が合格点以上**かつ**②全得点区分が基準点以上**の両条件を満たす必要がある。N1〜N3では言語知識・読解・聴解の各区分で**19点以上**、N4・N5では言語知識・読解で**38点以上**、聴解で**19点以上**が基準点となる。**基準点に達しない区分が1つでもあれば、総合得点がどれほど高くても不合格**となる。

この設計は「言語知識だけ優れている」「聴解だけ得意」といった偏った能力では合格できないことを意味し、バランスの取れた日本語能力の証明を目的とする。合格点自体は旧試験の合否判定水準との整合性を統計的に検証して設定されており、N1合格は旧1級合格、N4合格は旧3級合格に相当する水準である。

## 4. Can-do statements レベル別・技能別マトリクス

### Can-do statementsの構造と調査背景

JLPTのCan-do statementsは、**2010年9月〜2011年12月**に約65,000人の受験者を対象に実施された自己評価調査に基づく。各技能約30項目について「できる / 難しいがなんとかできる / あまりできない / できない」の4段階で回答を収集し、**合格ライン付近の合格者**の回答から各レベルの「できる」割合を推計した。JFスタンダード・CEFRの考え方を参考に開発され、「試験のシラバスではなく、合格者の能力を保証するものでもない」と明記されている。

### 聞く（Listening）Can-do項目

| レベル | 代表的なCan-do記述 |
|:------:|:------------------|
| **N1** | 政治・経済についてのテレビニュースの要点が理解できる / メディアで取り上げられる時事問題についての会話の概要が理解できる / フォーマルな場面でのスピーチの概要が理解できる |
| **N2** | 関心のあるテーマについての講演やスピーチの概要が理解できる / 学校や職場の会議に参加して議論の流れについていける / 標準的な日本語のテレビドラマや映画がだいたい理解できる |
| **N3** | 店で商品の説明を聞いて必要な情報を得ることができる / 駅やデパートでのアナウンスがだいたい理解できる / 周囲の人との雑談や自由な会話の概要が理解できる |
| **N4** | 身近な日常的話題（趣味、食べ物、週末の予定など）についての会話がだいたい理解できる / 簡単な指示を受けて何をすればよいか理解できる |
| **N5** | 店・郵便局・駅でよく使われるフレーズ（「いらっしゃいませ」「○○円です」など）が理解できる / 教室での先生や友達の簡単な自己紹介が理解できる |

### 読む（Reading）Can-do項目

| レベル | 代表的なCan-do記述 |
|:------:|:------------------|
| **N1** | 新聞・雑誌の政治・経済などの記事の要点が理解できる / 新聞などの社説の論点・意見・論の構成が理解できる / 小説を読んで登場人物の気持ちや話の流れが理解できる |
| **N2** | 丁寧な日本語で書かれた公式な手紙・メールの内容が理解できる / 取引先からの問い合わせや依頼の内容が理解できる / 一般的な国語辞典の語釈が理解できる |
| **N3** | 製品パンフレットから必要な情報（商品の特徴など）を得ることができる / 短編小説の話の筋が理解できる / 知人・友人からの葉書やメールが理解できる |
| **N4** | 新聞広告・チラシのセール日・価格が理解できる / 電車の時刻表・駅の案内表示を見て乗車時刻を確認できる |
| **N5** | イラスト付きの簡単な説明書（ゴミの出し方、料理の作り方）が理解できる / 学校などの予約表から自分の予約日時が理解できる |

### 話す・書く（Speaking/Writing）Can-do項目（JLPTでは直接測定されない）

| レベル | 話す | 書く |
|:------:|:-----|:-----|
| **N1** | 関心のあるテーマについての議論やディベートに参加し、論理的に意見を述べることができる | 論理的に意見を書くことができる |
| **N2** | 事前に準備すれば、自分の専門や詳しいテーマについてプレゼンテーションができる | 目上の知人宛てに基本的な敬語を使って手紙やメールを書くことができる |
| **N3** | 事前に準備すれば、自分の送別会などフォーマルな場面で短いスピーチができる | 自分の体験とそれについての感想を簡単に書くことができる |
| **N4** | 身近な日常的話題（趣味、週末の予定など）について会話に参加できる | 日記に短い文章を書くことができる |
| **N5** | 自己紹介をし、自分についての簡単な質問に答えることができる | 誕生日カードやお礼カードに短い文章を書くことができる |

### Can-do statementsの記述構造パターン

Can-doの文法構造は「**[条件・場面] + [対象・内容] + [行為] + ことができる**」の形式を取る。環境要因の記述方法として、**場所条件**（「駅で」「教室で」）、**対人条件**（「目上の知人に」「友人と」）、**準備条件**（「事前に準備すれば」）、**状況条件**（「フォーマルな場面で」「日常的な」）が用いられる。N5からN1に向けて、テキストの長さ、抽象度、処理速度、認知操作の複雑さ、社会言語能力（敬語の使い分け等）が段階的に高度化する設計となっている。

|:---------|:----------|:-----|
| **特定技能1号** | JLPT N4以上またはJFT-Basic A2以上 | 介護分野は追加で「介護日本語評価試験」合格が必要 |

### やさしい日本語とN3/N4レベルの関係

**やさしい日本語**は「難しい言葉を言い換えるなど、相手に配慮したわかりやすい日本語」と定義され、想定対象は**JLPT N3〜N4レベル**の外国人である。1995年の阪神・淡路大震災で外国人が必要な情報を得られなかった教訓から発展し、2020年に出入国在留管理庁・文化庁が「在留支援のためのやさしい日本語ガイドライン」を策定した。職場での活用では、一文を短くする、漢字にふりがなをつける、敬語は「です・ます」形を基本とする、といったポイントが示されている。

## 6. JLPTの妥当性研究と測定上の限界

### 測定しない能力と構造的制約

JLPTの最大の限界は**産出技能（話す・書く）を測定しない**ことである。試験は完全マークシート方式で、インタビュー・ロールプレイ・記述問題は一切含まれない。公式見解では「会話/話す能力」「作文/書く能力」のテスト追加計画はなく、CEFRの「産出活動」「やりとり」は評価対象外と明記されている。

N1/N2取得者の実際の産出能力には大きなばらつきがある。Hamabata (2019)の研究では、JLPT N2合格者8名をACTFL OPIで評価したところ、結果は**Intermediate MidからSuperiorまで**分布した。Can-do自己評価調査でも、N1合格者で「論理的に意見を書ける」と回答したのは**25-50%**にとどまる。

### 補完的評価方法

| 評価方法 | 測定技能 | JLPTとの関係 |
|:---------|:---------|:-------------|
| **BJT** | 聴解・読解（ビジネス場面） | 480点以上でN1相当、スコア制で細かい能力差を識別 |
| **OPI** | 話す能力 | JLPTで測定できないspeaking能力を直接評価 |
| **介護日本語評価試験** | 介護分野の日本語 | JLPT N4以上＋本試験で特定技能「介護」要件 |
| **J-TEST** | 読解・聴解＋記述 | 短文作成の記述問題あり、年6回実施 |

学術的には、Nishizawa et al. (2022)が「信頼性推定値の定期的公開」「より多くの妥当性検証研究」「JLPT得点と他評価の併用」を提言している。

## 7. SoE研究への示唆：Can-do statementsとケイパビリティアプローチの接続可能性

### 構造的類似性の分析

Can-do statementsとSen/Nussbaumのケイパビリティアプローチには、**「〜できる」という遂行可能性の記述様式**において構造的類似性が認められる。Senの「機能（functionings）」は「ある人が達成している状態や行為」を指し、「ケイパビリティ」は「達成可能な機能の集合」を意味する。Can-doの「政治・経済についてのテレビニュースの要点が理解できる」といった記述は、まさに「達成可能な状態」の具体化である。

重要な共通点は**環境・条件要因の明示**にある。Can-doでは「事前に準備すれば」「フォーマルな場面で」「ややゆっくりと話される会話であれば」といった条件が記述される。これはケイパビリティアプローチの「変換要因（conversion factors）」に相当し、同じ資源（言語知識）でも環境条件によって達成可能な機能が異なることを示す。純粋な「能力」ではなく「特定条件下での遂行可能性」を記述する点で、両者は**欠損モデルではなく環境因子モデル**（ICF的視点）と親和性を持つ。

### 本人主権型アセスメントへの応用考察

Can-do自己評価の方法論は、**本人による能力の自己認識と外部評価の統合**という点でInput Constitutional AIフレームワークの理論的基盤に示唆を与える。JLPTでは試験得点という「外部評価」とCan-do自己評価という「本人評価」の相関を調査し、両者のギャップを可視化している。これは障害福祉就労支援における「本人が認識する『できること』」と「環境条件を含めた『実際にできること』」の関係性分析に応用可能である。

ただし限界も存在する。Can-doは**受容技能中心**で設計されており、「話す」「書く」の自己評価は試験得点との直接的検証ができない。また、Can-doの条件記述は**一般化された場面**（「店で」「職場で」）であり、個別具体的な環境因子（職場の物理的環境、対人関係、支援体制など）は含まれない。本人主権型アセスメントでは、これらの**個別化された環境因子**をどう記述・評価するかが課題となる。

Can-doの「段階性」（N5→N1の発達的序列）は、ケイパビリティの「程度」を評価する枠組みとして参考になるが、福祉分野では必ずしも「より高度な機能＝より良い」とは限らない。**本人にとっての「十分な」機能レベル**を誰がどう判断するかという規範的問題は、言語能力評価とは異なる倫理的検討を要する。
