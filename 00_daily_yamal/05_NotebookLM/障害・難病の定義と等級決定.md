# **日本における3障がい（身体・療育・精神）及び難病の定義と等級判定基準に関する総合的考察**

## **序論：日本における障害者福祉制度のパラダイムと本稿の目的**

現代日本の社会保障体系において、障害を持つ人々や難病に罹患した人々に対する公的支援は、極めて複雑かつ多層的な法的枠組みによって構成されている。一般に「3障がい」と総称される身体障害、知的障害（療育手帳の対象）、および精神障害（発達障害を含む）は、それぞれ異なる歴史的背景と法律に基づいて制度化されてきた。加えて、原因が不明で治療法が未確立な「指定難病」に対する支援制度も、障害福祉とは別の次元から発展し、近年では両者の境界が交差する場面も増加している。

これらの支援制度の根幹をなすのが、各領域における「定義」と「等級（級数）または重症度の決定方法」である。日本の制度は、単に医学的な疾患名が存在することをもって公的支援の対象とするのではなく、その疾患や機能障害が「日常生活や社会生活にどのような、そしてどの程度の制限をもたらしているか」という能力障害の観点を厳密に評価する。この評価基準は、従来の医学的な欠損に焦点を当てる「医学モデル」から、環境と個人の相互作用によって生じる生活上の障壁を評価する「社会モデル（国際生活機能分類：ICFの理念）」へと歴史的に移行しつつあるが、領域ごとにその反映度合いは大きく異なる。

本稿では、身体障害、知的障害（療育）、精神障害（発達障害を含む）、および難病の4つの領域について、それぞれの法的定義、障害等級や重症度の決定メカニズムを、厚生労働省等が公開している一次情報のURLとともに網羅的に整理する。さらに、表層的な判定基準の羅列にとどまらず、それらの基準が実務上どのように解釈され、社会的にどのような意味を持つのかについて、多角的な視点から精緻な考察を展開する。

## **第1章：精神障害および発達障害の定義と判定基準**

日本の精神障害者福祉は、「精神保健及び精神障害者福祉に関する法律（精神保健福祉法）」を基盤として運用されている。精神障害者保健福祉手帳制度は、精神疾患によって長期にわたり日常生活または社会生活への制限を受ける者を対象としており、単一の検査数値で測ることが困難な「心と脳の機能障害」を、社会生活能力というレンズを通して評価する点に大きな特徴がある。

この判定基準および運用の詳細に関する一次情報は、厚生労働省の公式文書「精神障害者保健福祉手帳の障害等級の判定基準について（平成07年09月12日健医発第1133号）」（URL: [https://www.mhlw.go.jp/web/t\_doc?dataId=00ta4615\&dataType=1\&pageNo=1](https://www.mhlw.go.jp/web/t_doc?dataId=00ta4615&dataType=1&pageNo=1)）および「精神障害者保健福祉手帳の障害等級の判定基準の運用に当たって留意すべき事項について（平成07年09月12日健医精発第46号）」（URL: [https://www.mhlw.go.jp/web/t\_doc?dataId=00ta4618\&dataType=1\&pageNo=1](https://www.mhlw.go.jp/web/t_doc?dataId=00ta4618&dataType=1&pageNo=1)）に極めて詳細に規定されている1。

### **精神障害の等級判定における基本構造**

精神障害の等級判定は、医師の主観に過度に依存することを防ぎ、全国的な公平性を担保するため、以下の4つの連続した手順を経て総合的に行われる1。

1. **精神疾患の存在の確認**: ICD-10（国際疾病分類）に基づく精神疾患の診断が確定しているかを確認する。  
2. **精神疾患（機能障害）の状態の確認**: その疾患が現在どのような症状を呈し、どの程度の深刻さ（高度、中等度、軽度など）にあるかを評価する。  
3. **能力障害（活動制限）の状態の確認**: 疾患に起因して、実際の生活場面でどのような具体的な支障が生じているかを評価する。  
4. **精神障害の程度の総合判定**: 上記2と3を掛け合わせ、最終的な等級（1級〜3級）を決定する。

### **能力障害（活動制限）を測る8つの指標とその社会的含意**

精神障害判定の核心となるのが「能力障害（活動制限）の状態」の評価である。厚生労働省は、患者の日常生活自立度を測るため、以下の8つの具体的な生活能力項目を設定している1。これらの項目は、単なる生命維持のレベルを超え、現代社会において独立した市民として生活するために不可欠な社会的機能の網羅的リストとなっている。

| 評価項目 | 具体的な状態の定義と社会的含意 |
| :---- | :---- |
| **1\. 適切な食事摂取** | 調和のとれた適切な食事が自発的にできるか。栄養失調の回避だけでなく、生活リズムの維持能力を測る1。 |
| **2\. 身辺の清潔保持** | 洗面、入浴、更衣、清掃等が自発的に可能か。これは就労や対人関係構築の前提となる社会性の基本指標である1。 |
| **3\. 金銭管理・買物** | 金銭管理能力があり、計画的で適切な買物ができるか。認知機能の低下や躁状態での浪費等、経済的自立を阻む要因を評価する1。 |
| **4\. 通院・服薬の規則性** | 通院や服薬を必要とする場合、規則的に行うことができるか。自己の病識（インサイト）と健康管理能力の有無を示す1。 |
| **5\. 意思伝達と対人関係** | 家族や知人・近隣等と適切な意思伝達ができ、協調的な対人関係を作れるか。孤立のリスクと社会的ネットワークの維持能力を測る1。 |
| **6\. 安全保持と危機対応** | 身辺の安全を保持し、危機的状況に適切に対応できるか。自己判断能力の欠如による生命・財産へのリスク評価である1。 |
| **7\. 社会的手続と施設利用** | 社会的手続（役所への申請等）を行ったり、一般の公共施設を利用することができるか。社会の制度的インフラへのアクセス能力を測る1。 |
| **8\. 文化的社会的活動への参加** | 社会情勢や趣味・娯楽に関心を持ち、活動に参加できるか。精神的エネルギーの枯渇（陰性症状等）や抑うつ状態の深さを示す1。 |

判定にあたっては、これらの項目が「自発的に行えるか」「援助（声かけ、見守り、直接的な介助など）を必要とするか」「援助があっても困難か」というグラデーションで評価される1。

### **各等級（1級〜3級）の判定基準の精緻な解釈**

精神疾患の重症度と、前述した8項目の能力障害の度合いを組み合わせることで、最終的な等級が決定される1。手帳制度における各等級は、単なる数字の違いではなく、福祉サービスの受給権（税面の控除額、各種料金の免除、生活保護の障害者加算など）に直結する決定的な境界線である。

| 等級 | 精神疾患（機能障害）の状態の基準 | 能力障害（活動制限）の基準 | 総合的な状態像の解釈 |
| :---- | :---- | :---- | :---- |
| **1級** | 統合失調症の「高度」な残遺状態や思考障害、気分障害の「高度」な病相期の持続、非定型精神病の高度な残遺状態、てんかんの頻発する発作等1。 | 日常生活の用を弁ずることを「不能」ならしめる程度。前記8項目のうち複数で「援助なしにはできない」状態であり、常時の援助が必須1。 | 他者の全面的な介入がなければ、地域社会での生存そのものが危ぶまれる極めて重篤な状態。入院環境から地域移行する際の最大の支援対象となる。 |
| **2級** | 統合失調症の人格変化や異常体験、器質性精神障害（記憶障害、遂行機能障害等）のうち1つ以上が「中等度」等1。 | 日常生活に「著しい制限」を受ける程度。8項目について自発的には行えず、「援助なしにはできない」状態が複数認められる1。 | 単独での生活は著しく困難であり、グループホームや訪問看護等の継続的な公的・私的サポートを要する状態。 |
| **3級** | 統合失調症の著しくはない思考障害等、器質性精神障害がいずれも「軽度」、非定型精神病の残遺状態がこれらに準ずるもの等1。 | 日常生活または社会生活に「制限」を受ける程度。8項目について「自発的に行うことはできるが、なお援助を必要とする」または不安定1。 | 単身生活や就労（障害者雇用など）は可能である場合が多いが、ストレス下での状態悪化を防ぐため、職場での配慮や定期的な通院・見守りを必要とする状態。 |

精神疾患の分類として、器質性精神障害（ICD-10のF04、F06、F07等に該当）は、外傷性脳損傷や脳血管障害等による脳の器質的病変を原因とし、記憶障害、注意障害、遂行機能障害、社会的行動障害などの認知障害を引き起こすものを指す3。非定型精神病は、発病が急激で周期性の経過を示し予後が良いという特性を持つため、その時点の残遺状態の程度によって厳格に判定される3。また、中毒精神病（アルコールや薬物等）については、認知症の進行度やその他の精神神経症状の強さが等級を左右する重要な指標となる1。

### **発達障害の特例的な位置づけと「二次障害」の評価**

日本の法体系における顕著な特徴として、発達障害（自閉スペクトラム症、注意欠如・多動症、学習障害など）が単独の「発達障害者手帳」を持たず、精神保健福祉法の枠組みである精神障害者保健福祉手帳の中で評価される点が挙げられる1。発達障害者支援法に基づく概念が、手帳等級判定においては独自の解釈ロジックで運用されている。

一次情報（URL: [https://www.mhlw.go.jp/web/t\_doc?dataId=00ta4615\&dataType=1\&pageNo=1](https://www.mhlw.go.jp/web/t_doc?dataId=00ta4615&dataType=1&pageNo=1)）によれば、発達障害の判定は「主症状（発達障害に生来的に特有の症状）」と「その他の精神神経症状（二次的なうつ病、適応障害、パニック障害など）」の2つの軸の掛け合わせで評価される1。

* **1級:** その主症状とその他の精神神経症状が「高度」のもの1。  
* **2級:** その主症状が「高度」であり、その他の精神神経症状が「ある」もの1。  
* **3級:** その主症状とその他の精神神経症状が「ある」もの1。

この記述から導き出される重要なインサイトは、発達障害そのものの特性（主症状）だけではなく、社会や学校、職場環境とのミスマッチによって後天的に引き起こされる「二次障害（その他の精神神経症状）」が、等級を決定する決定的な要因として機能しているという事実である1。これは、本人の特性に対する適切な環境調整（合理的配慮）が行われ、二次障害が軽減された場合、皮肉にも手帳の等級が下がる（あるいは非該当となる）可能性を内包している。能力障害を環境との相互作用で捉える社会モデル的評価の現れであると同時に、支援が充実すれば制度的保護から外れるという福祉制度のパラドックスを示唆している。

### **てんかんの等級判定：発作の不確実性と生活制限の定量化**

てんかんの判定は、精神障害の中でも極めて特異で定量的な運用マトリクスを持っている（一次情報URL: [https://www.mhlw.go.jp/web/t\_doc?dataId=00ta4618\&dataType=1\&pageNo=1](https://www.mhlw.go.jp/web/t_doc?dataId=00ta4618&dataType=1&pageNo=1)）2。てんかんは、発作間欠期（発作が起きていない平常時）には全く健常と変わらない生活を送れる患者が多い一方、発作の予測不可能性が就労制限（運転免許の取得制限や危険作業の禁止等）や社会参加への甚大な障壁となる。

そのため、厚生労働省の基準では、発作のタイプ（意識障害の有無、転倒の有無等によりイ、ロ、ハ、ニ等に分類）と「発作の頻度」、そして「日常生活能力の程度」をクロス集計して等級を算出する2。

| 発作のタイプと発生頻度 | 日常生活能力の制限程度 | 該当する障害等級の目安 |
| :---- | :---- | :---- |
| **ハ、ニの発作が月に1回以上ある場合** | (4) 日常生活に著しい制限を受け、常時援助を必要とする、または (5) 身の回りのことがほとんどできない | **おおむね1級程度** 2 |
| **イ、ロの発作が月に1回以上**、または **ハ、ニの発作が年に2回以上** | (3) 日常生活に著しい制限を受け、時に応じて援助を必要とする | **おおむね2級程度** 2 |
| **イ、ロの発作が月に1回未満**、または **ハ、ニの発作が年に2回未満** | (2) 日常生活又は社会生活に一定の制限を受ける | **おおむね3級程度** 2 |

頻繁に繰り返す発作、または発作に起因する知能障害・精神神経症状が高度である場合は1級と見なされる1。このマトリクスは、医学的な「症状の発生確率」を、直接的に社会的な「活動制限のリスク」として換算する優れた制度的発明であると言える。発作が月に1回未満であっても、社会生活に一定の制限がある限り3級の保護対象となることは、てんかんという疾患の社会的スティグマに対する制度的配慮を反映している。

## **第2章：身体障害の定義と身体障害者障害程度等級表**

身体障害者福祉法に基づく身体障害者手帳制度は、日本の障害者福祉における最も歴史の古い根幹的制度である。精神障害が「生活能力の制限」という社会モデルに立脚しているのとは対照的に、身体障害の判定は極めて解剖学的、生理学的、かつ定量的な「医学（生物医学）モデル」に深く根ざしている。

対象となるのは、視覚、聴覚・平衡機能、音声・言語・そしゃく機能、肢体不自由（上肢、下肢、体幹、乳幼児期以前の非進行性の脳病変による運動機能障害）、および内部機能（心臓、じん臓、呼吸器、ぼうこう・直腸、小腸、ヒト免疫不全ウイルスによる免疫機能、肝臓）に「永続的な障害」がある者である4。この「永続性」の要件により、治療によって回復が見込まれる一時的な機能低下は対象外となる。

詳細な判定基準となる「身体障害者障害程度等級表」は、厚生労働省の公式資料（一次情報URL: [https://www.mhlw.go.jp/content/0000172197.pdf](https://www.mhlw.go.jp/content/0000172197.pdf) 等）にて規定されており、身体の部位や臓器ごとにセンチメートル単位、角度単位、あるいは点数単位での厳密な数値規定が設けられている5。

### **等級の全体構造と「累積的評価」のメカニズム**

障害の程度は、最も重度である1級から最も軽度の7級までに区分される4。しかし、法制度の運用上、大きな特徴が存在する。身体障害者福祉法において単独で手帳交付の対象となるのは「1級から6級まで」である4。7級の障害は、それ単独では手帳交付の要件を満たさない。 ただし、日本の等級制度には「累積ルール（重複障害の取り扱い）」が存在する。7級に該当する障害が2つ以上重複する場合や、6級以上の障害と7級の障害が重複する場合には、それらを合算して上位の等級（この場合は6級等）として認定される4。このメカニズムは、個別の機能低下は軽微であっても、複数が重なることで日常生活に及ぼす相乗的な障壁を評価する試みとして解釈できる。

各領域の等級の範囲は以下の通りである。

* **視覚障がい**: 1級〜6級4  
* **音声・言語・そしゃく機能障がい**: 3級〜4級4  
* **肢体不自由（上肢・下肢）**: 1級〜7級（手帳交付対象は実質6級まで）4  
* **体幹・脳病変による運動機能障がい**: 1級〜5級（脳病変は1〜7級まで）4  
* **内部障がい（心臓、じん臓等）**: 1級〜4級（疾患によっては2級が存在しないものもある）4

### **視覚障害の定量的かつ多面的な判定基準**

視覚障害の判定は、単なる「見えにくさ」という主観的訴えを排除し、万国式試視力表による「矯正視力」と、精密な視野測定機器による「視野狭窄」の測定結果によって厳密に定義される5。 例えば、身体障害の5級に該当する視覚障害の認定基準は、以下の複数の客観的数値のいずれかを満たすことと規定されている5。

1. **視力による判定:** 視力の良い方の眼の視力が0.2かつ他方の眼の視力が0.02以下のもの。  
2. **視野欠損による判定:** 両眼による視野の2分の1以上が欠けているもの。  
3. **中心視野角度による判定:** 両眼中心視野角度が56度以下のもの。  
4. **開放視認点数による判定:** 両眼開放視認点数が70点を超えかつ100点以下のもの。  
5. **中心視野視認点数による判定:** 両眼中心視野視認点数が40点以下のもの。

ここから読み取れるインサイトは、現代社会における視覚情報の重要性が多様化していることに対応し、中心部の視力だけでなく、歩行等の安全確認に不可欠な「視野の広さ」や「視認点数」という多角的な計測指標が行政基準に明文化されていることである5。この厳格な数値主義は、行政決定の透明性と公平性を担保する強力な武器となる反面、眼精疲労の極度な悪化や光覚異常（極端な眩しさ）といった数値化しにくい主観的苦痛が等閑視されやすいという構造的限界も内包している。

### **肢体不自由と音声・言語・そしゃく機能の解剖学的規定**

肢体不自由に関しても、解剖学的な欠損部位や関節の可動域に基づく詳細極まりない規定が存在する。

下肢機能障害の5級基準を例にとると、以下のように定義されている4。

* 一下肢の股関節または膝関節の機能の「著しい障がい」  
* 一下肢の足関節の機能を「全廃」したもの  
* 一下肢が健側に比して「5cm以上」または「健側の長さの15分の1以上」短いもの

ここで特筆すべきは、「5cm」という具体的な物理的長さが等級を決定する絶対的な分水嶺となっている点である4。脚長差が5cmに達すると、代償的な骨盤の傾斜だけでは歩行のバランスを保つことが困難となり、特注の補装具や杖が必須となるというバイオメカニクス的知見が、そのまま法規定へと変換されている。

上肢機能障害においては、手の「巧緻性（細かい作業を行う能力）」の喪失が重く見られる。6級の基準として「一上肢のおや指の機能の著しい障害」や「ひとさし指を含めて一上肢の二指を欠くもの」、または「ひとさし指を含めて一上肢の二指の機能をほとんど失っているもの」が規定されている6。母指（おや指）の対立運動が人間の道具操作の根幹であるという医学的事実に基づき、たった一指の障害であっても独立した等級として評価されている6。

一方で、音声機能、言語機能、そしゃく機能の障害に対する等級規定は、極めてシンプルかつ二元的なアプローチをとっている。

* **3級:** 音声機能、言語機能またはそしゃく機能の「喪失」4  
* **4級:** 音声機能、言語機能またはそしゃく機能の「著しい障害」4

この領域では1級や2級が存在しない6。これは、発声や嚥下機能の喪失が患者のQOL（生活の質）に甚大な影響を与えるにもかかわらず、手足の完全麻痺や全盲と比較して、社会における「労働能力の完全な喪失」とはみなされないという、制定当時の社会的な労働観・障害観が現在も残存していることを示唆している。

### **内部機能障害：病気と障害の境界線**

内部障がいは、外見からは認知されにくい「見えない障害」の代表格である。心臓、じん臓、肝臓、免疫機能（HIV）などの臓器不全が対象となる。 規定文書において、内部障がいは「（各臓器）の機能の障害により、社会での日常生活活動が著しく制限されるもの」と定義されている5。例えば、ヒト免疫不全ウイルスによる免疫機能障害や、肝臓機能障害もこの文言で定義されている5。

ここにおける最大の論点は、内部障がいが「疾患の慢性的な進行状態」と「永続的な機能障害」の境界線上に位置していることである。身体内部の臓器機能低下は、強烈な「易疲労性（疲れやすさ）」や「消耗」を引き起こす。これは精神障害判定における「能力障害」の概念と実質的に重なり合う部分が大きいが、身体障害の枠組みにおいては、あくまで血液検査の数値や心電図、呼吸機能検査といったバイオマーカーの異常値が要件として先行する。数値基準を満たさなければ、どれほど深刻な倦怠感により就労が困難であっても、身体障害者手帳の対象とはならないという厳格な線引きが存在する。

## **第3章：知的障害（療育手帳）の定義と自治体裁量による判定運用**

日本における3障がいのうち、身体障害と精神障害がそれぞれ独立した法律（身体障害者福祉法、精神保健福祉法）に基づいて手帳制度を全国一律で厳格に規定しているのに対し、知的障害者を対象とする「療育手帳制度」は、全く異なる法的成り立ちを持つ。 療育手帳は法律の条文に直接的な根拠を持つ手帳ではなく、昭和48年（1973年）に発出された「厚生省事務次官通知」を根拠として創設された行政指導に基づく制度である7。

この根本的な一次情報となるのが、「昭和48年9月27日厚生省発児第156号（平成3年一部改正）」に基づく「療育手帳制度要綱」である（関連URL: [https://www.cao.go.jp/bunken-suishin/teianbosyu/doc/tb\_h30fu\_12mhlw\_180\_2.pdf](https://www.cao.go.jp/bunken-suishin/teianbosyu/doc/tb_h30fu_12mhlw_180_2.pdf) および [https://www.mhlw.go.jp/content/12200000/000639205.pdf](https://www.mhlw.go.jp/content/12200000/000639205.pdf)）7。

### **通知に基づく非法定制度と自治体裁量の意義**

法律による全国一律の等級規定が存在しないという事実は、日本の知的障害福祉における最大の特徴であり、同時に最大の議論の的でもある。療育手帳の判定基準、呼称、さらには等級の区分数に至るまで、各都道府県および政令指定都市、中核市等の自治体の裁量に大きく委ねられている。 制度要綱第2項において、手帳の交付の決定主体は「都道府県知事等（中核市の長を含む）」であると明記されている7。

判定の実務を担うのは、医療機関の一般の医師ではなく、公的な専門機関である。

* **18歳未満の児童:** 児童相談所が判定を行う7。  
* **18歳以上の成人:** 知的障害者更生相談所が判定を行う7。

これらの機関に所属する児童心理司や判定員が、標準化された知能検査（田中ビネー知能検査、WISC等）による知能指数（IQ）の測定結果と、面接や保護者からのヒアリングに基づく「日常生活における自立度（基本的生活習慣、対人関係、社会適応能力等）」の評価を総合し、知的障害に該当するか否かを決定する7。IQが一定の数値（例えばIQ70や75）を下回ることが一般的な目安とされるが、IQ数値が高くても社会適応能力が著しく低い場合は交付の対象となるケースもあり、専門職の裁量権が大きく認められている。

### **障害程度の区分：「A」と「B」の包括的分類**

国が指針として示す標準的な療育手帳制度要綱においては、障害の程度を示す記号として、極めてシンプルな分類が提示されている。手帳の判定欄には「A」または「B」の記号が記載される8。

* **A区分:** 障害の程度が「重度」であることを意味する8。  
* **B区分:** 重度「それ以外」（中度・軽度など）を意味する8。

この手帳に記載された「A」「B」の記号は、電車、バス、飛行機などの交通機関を割引運賃で利用する際や、税金の控除、各種福祉サービスを受給する際の決定的な証明証となるため、乗車中も常に携行することが要綱上でも求められている8。

しかし実際には、自治体はこの国の「A・B」という大枠を基に、よりきめ細かい独自の細分化を行っている。例えば、A1（最重度）、A2（重度）、B1（中度）、B2（軽度）といった4段階評価を採用する自治体もあれば、東京都のように「愛の手帳」という独自の名称を用い、1度から4度という区分を用いている自治体もある。岡山県等のように独自の判定基準を公開している自治体も存在する（関連URL: [https://www.pref.okayama.jp/page/280806.html](https://www.pref.okayama.jp/page/280806.html) ※該当リンクは現在アクセス不能となっているが、各都道府県が独自にウェブサイト上で基準を公開する運用が一般的である）9。この自治体間の基準の不統一は、患者が引っ越し等の理由で他県へ転出入した際に、再判定によって等級が変わる、あるいは手帳の対象外となるといった「制度の谷間」を生み出す原因としてしばしば指摘されている。

### **動的評価システム：「交付後の障害の程度の確認（再判定）」**

療育手帳制度におけるもう一つの決定的な特徴は、障害を固定化された永続的なものとは見なさない点にある。発達過程にある児童はもちろんのこと、成人であっても、知的機能や社会適応能力は、周囲の教育環境、療育訓練、就労支援などの環境要因との相互作用によって変化（向上または低下）する可能性を秘めている。

そのため、療育手帳制度要綱第6項「交付後の障害の程度の確認」においては、「都道府県知事等は、手帳の交付後、手帳の交付を受けた知的障害者の障害の程度を確認するため、原則として2年ごとに児童相談所又は知的障害者更生相談所において判定を行うものとする」と明記されている7。

患者に交付される手帳には「次の判定年月」という欄が設けられており、保護者や本人はこの期限までに指定された相談所で再判定を受けなければならない8。もしこれを行わなければ、手帳の効力が停止し、付随するサービスを利用できなくなる旨が手帳の注意事項として記載されている8。 この「2年ごとの再判定を原則とする」という規定7は、福祉予算の適正化という行政側の要請に加えて、人間の発達可能性を肯定し、その時点の能力に最も適した支援を提供するという「療育的観点（エデュケーショナル・モデル）」が強く反映されたシステムであると結論づけることができる。

## **第4章：難病（指定難病）の定義と重症度分類に基づく医療費助成判定**

これまでに詳述した3障がい（身体・知的・精神）の制度は、いずれも「障害者手帳」を発行し、生活全般にわたる総合的な福祉サービスや社会的配慮を提供することを目的としている。これに対し、難病（指定難病）に対する公的支援制度は、「難病の患者に対する医療等に関する法律（難病法）」という独立した法律を根拠とし、そのアプローチも目的も根本的に異なる（一次情報URL: [https://laws.e-gov.go.jp/law/426CO0000000358](https://laws.e-gov.go.jp/law/426CO0000000358)）10。

難病制度の主たる目的は、原因が究明されておらず、効果的な治療方法が未確立であり、かつ長期の療養を必要とする希少疾病に対して、「医療費の自己負担を軽減すること（医療費助成）」と、患者データを集約し治療研究を推進することにある。

### **難病法と指定難病の法的枠組みおよび負担軽減の構造**

難病法において、すべての難病が助成の対象となるわけではなく、国が政令で定める要件を満たした「指定難病」に罹患していることが前提となる。この制度は、健康保険法や身体障害者福祉法等の既存の社会保障制度の枠組みの上に、追加的な防波堤として構築されている10。難病医療法施行令等において、これらの法律との適用関係が厳密に定められている10。

医療費の自己負担上限額は、患者世帯の所得水準に応じて階層化されている。法令の規定によれば、月額の負担上限額は所得階層によって「2万円」や「3万円」などに設定されており、例えば所得が「80万円以下である者」に対する規定や、支給認定を受けた指定難病の患者が「重症認定患者」である場合における詳細な負担免除・軽減規定が存在する10。これにより、終わりの見えない長期療養がもたらす家計の破綻を防ぐセーフティネットとして機能している。

### **医療費助成の要件：重症度分類と機能的評価基準**

指定難病に罹患している（病名診断が確定している）という医学的事実だけでは、直ちに医療費助成（特定医療費受給者証の交付）の対象とはならない。助成の申請が承認されるためには、疾患ごとに定められた「重症度分類」等に照らし合わせ、病態が一定水準以上の重症状態にあると判定される必要がある（関連一次情報URL: [https://www.mhlw.go.jp/file/05-Shingikai-10601000-Daijinkanboukouseikagakuka-Kouseikagakuka/0000184562.pdf](https://www.mhlw.go.jp/file/05-Shingikai-10601000-Daijinkanboukouseikagakuka-Kouseikagakuka/0000184562.pdf) および [https://www.nanbyou.or.jp/wp-content/uploads/2024/12/R061227\_kaisei01.pdf](https://www.nanbyou.or.jp/wp-content/uploads/2024/12/R061227_kaisei01.pdf)）11。

厚生労働省が定める重症度分類の運用原則は、疾患の多様性に対応するため、極めて柔軟かつ実践的な階層構造を持っている11。

1. **疾患固有の重症度分類の原則適用:** 疾病ごとに医学会等が作成した固有の重症度分類等が存在する場合は、原則として当該分類を用いる11。  
2. **臓器別・機能別分類の代替適用:** 疾患固有の分類において、日常生活や社会生活への支障の程度が測りにくい場合、あるいは明確な分類が存在しない場合は、心、肺、肝、腎、視力、聴力といった「臓器領域等ごとに作成されている重症度分類」や、「ADL（Activities of Daily Living：日常生活動作）評価」を疾患の特性に応じて代用する11。  
3. **診断基準自体の準用:** 家族性高コレステロール血症（ホモ接合体）などのように、段階的な重症度分類を設けることが困難であっても、その「診断基準を満たすこと自体」が、概ね日常生活または社会生活への著しい支障を表していると医学的に認められる疾病については、特例的に当該診断基準そのものを重症度分類として代用する11。

機能的評価の代表的な指標として「Barthel Index（バーセルインデックス）」が多用される。これは、食事、車いすからベッドへの移動、整容、トイレ動作、入浴、歩行、階段昇降、着替え、排便・排尿コントロールの各項目を採点し、患者の自立度を測る世界的指標である。指定難病の重症度基準において、「Barthel Indexが85点以下を対象とする」といった形で、明確なカットオフ値が設定されている疾病が存在する12。この「85点」という数値は、何らかの生活行為において物理的な介助や見守りが必要なレベルを示しており、ここでも「医学的欠損」と「社会的・生活的支障」を接続する試みが見て取れる。

### **臨床現場の実態に即した評価ルールの精緻化**

指定難病の判定において最も注目すべきは、臨床現場における症状の「時間的変動」に対する制度的配慮である。診断基準および重症度分類の適応における留意事項として、以下の極めて重要なルールが明文化されている12。

* **診断時のデータ要件:** 病名診断に用いる臨床症状や検査所見等に関して、診断基準上に特段の規定がない場合には、「いずれの時期のものを用いても差し支えない（ただし、当該疾病の経過を示す臨床症状等であって、確認可能なものに限る）」とされている12。これは、希少疾患ゆえに過去の検査記録を遡らざるを得ない患者の立証負担を軽減する措置である。  
* **「直近6か月の最悪状態」の採用:** 治療開始後における重症度分類については、「適切な医学的管理の下で治療が行われている状態であって、直近6か月間で最も悪い状態を医師が判断することとする」と規定されている12。多くの難病は寛解と増悪（フレア）を繰り返す。一時的にステロイド等の投薬で症状が抑え込まれている「その日」の状態で判定を行えば、根本的な病態の重篤さを過小評価してしまう。この「直近6か月の最悪状態」ルールは、変動する疾患特性を捉える極めて優れた運用上の知恵である。

### **「軽症者特例」と高額医療継続の経済的救済**

難病医療費助成の制度設計が、純粋な医学的重症度だけでなく「経済的側面」を多分に考慮していることを決定的に示すのが、「高額な医療を継続することが必要な者」への特例措置である12。 法令および通知において、「症状の程度が上記の重症度分類等で一定以上に該当しない者であるが、高額な医療を継続することが必要なものについては、医療費助成の対象とする」と明記されている12。いわゆる「軽症者特例」である。

難病医療においては、極めて高価なバイオ医薬品や分子標的薬を継続的に投与することで、かろうじて日常生活を維持（重症化を予防）している患者が多数存在する。もし「現在は症状が軽いから」という理由で助成を打ち切れば、薬価の自己負担に耐え切れず治療を中断し、結果として重症化し、より巨大な医療費・社会的コストを発生させるという悪循環に陥る。この特例は、難病法の本質が「医学的な重症度の証明」に対する見返りではなく、「経済的障壁による治療機会の喪失の阻止」にあることを証明している。

### **疾患固有の判定基準事例：LMNB1関連大脳白質脳症**

指定要件がいかに高度で専門的な医学的知見に基づいているかを示す一例として、「LMNB1関連大脳白質脳症」の基準が挙げられる12。 診断基準において、大脳白質病変は「初期には散在性のこともあるが、やがて融合し、左右対称性の白質変化を呈する」「前頭葉・頭頂葉優位で、側脳室近傍の白質は保たれる傾向がある」「中小脳脚病変は病初期から信号変化を認めることが多いが、信号変化の程度は症例間での差が大きい」といった、MRI等の画像診断における微細な所見が徹底的に言語化されている12。さらに、「延髄から頚髄にかけての錐体路病変を認める症例も存在する」といった例外的事象や、「家族歴が明らかでないde novo変異と考えられる孤発例が少数報告されているが、ほとんどは常染色体顕性（優性）遺伝形式であり、浸透率はほぼ100%と考えられている」といった遺伝学的知見も組み込まれている12。

これほどまでに精緻な医学的診断によるスクリーニングを通過した上で、初めて前述の「Barthel Index 85点以下」などの機能的・社会的評価が重畳され、助成対象が確定するという、二段構えの厳格なゲートキーピングが行われているのである。

## **第5章：制度横断的な分析と今後の展望（結語）**

これまで詳細に分析してきたように、日本における3障がい（身体、療育、精神・発達）および難病の定義と判定基準は、それぞれが依拠する歴史的文脈と政策的目的の違いから、全く異なる哲学に基づいて設計され、運用されている。

1. **身体障害判定**は、解剖学的な欠損や生理的機能低下をセンチメートルや角度といった単位で定量化する「静的・医学的・部位特化型モデル」である4。客観性と透明性に優れる反面、数値化しにくい主観的苦痛（極度の疲労、痛み）や環境要因との相互作用による生活のしづらさを評価しきれないという限界を持つ。  
2. \*\*精神障害判定（発達障害を含む）\*\*は、疾患の存在を前提としつつも、それが食事や対人関係等の8つの日常生活項目において「どれほどの援助を必要とするか」を評価する「動的・機能的・社会モデル的アプローチ」を採用している1。環境調整が進めば等級が下がる可能性を内包するほど、個人と環境の相互作用を重視している1。  
3. \*\*知的障害判定（療育手帳）\*\*は、法律ではなく通知に基づく行政指導という特異な形態をとり、知能指数と社会適応能力を組み合わせた専門職の総合判定に委ねられている7。2年ごとの再判定を原則とする等、発達可能性を前提とした「エデュケーショナル・モデル」の側面が強いが、自治体間の基準の不統一が課題である7。  
4. **指定難病判定**は、原因不明の希少疾病に対する医療費助成に特化し、病態の「直近6か月の最悪状態」の評価や、高額医療継続による軽症者特例など、流動的で長引く治療生活の実態に即した「医療・経済的救済モデル」である11。

これら4つの巨大なシステムが並立している日本の現状は、対象者の多様なニーズに対し、世界的に見ても極めて精緻できめ細やかな安全網を提供していると評価できる。しかし同時に、その「縦割り」の構造ゆえに、どの制度の定義にも完全には合致しないが生活に困窮しているという「制度の谷間」が生じやすいという構造的欠陥も抱えている。

例えば、難病によって極度の易疲労性があり就労できないが、重症度分類の基準を満たさず、かつ臓器の数値異常が身体障害等級表の基準にも達しない患者は、公的な福祉サービスの対象から完全に外れてしまうリスクがある。

今後、日本の社会保障制度が直面する最大の課題は、疾患名や障害の部位に縛られた縦割りの評価基準から脱却し、ICF（国際生活機能分類）が提唱するように、「疾患を問わず、現状の生活機能がどの程度制限されており、社会参加にあたってどのような環境的障壁が存在するか」という横断的かつ普遍的な評価軸を構築・融合していくことである。

本稿で提示した厚生労働省の一次情報に基づく各制度の厳格な定義と運用ルールの理解は、その複雑な体系を解きほぐし、真に当事者の生活実態に適合する未来の制度設計を論じるための不可欠な出発点となる。
