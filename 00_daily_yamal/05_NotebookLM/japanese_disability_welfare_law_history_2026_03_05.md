## はじめに：なぜ法制度の変遷を知る必要があるのか

日本の障がい福祉法制度は、戦後80年の間に根本的な転換を繰り返してきました。「保護の対象」から「権利の主体」へ、「措置制度」から「契約制度」へ、「施設収容」から「地域生活」へ。この変遷は単なる法律の改正ではなく、「障害とは何か」「支援とは何か」という根本的な問いに対する社会の答えが変わってきた歴史です。

現場の支援者がこの変遷を理解することで、今の制度がなぜこのような形になっているのか、どこに課題が残っているのか、そしてこれからどこに向かうべきなのかが見えてきます。

---

## 第1章：戦後復興期 ― 福祉三法の時代（1945〜1960年代）

### 1.1 戦後の出発点：日本国憲法と福祉の基盤

1946年に公布された日本国憲法は、第25条で「すべて国民は、健康で文化的な最低限度の生活を営む権利を有する」と規定しました。この生存権規定が、戦後の障がい福祉法制度の出発点となります。

### 1.2 福祉三法の成立

戦後の混乱期、障がい者を含む社会的弱者への緊急的対応として、以下の法律が制定されました。

| 法律名 | 制定年 | 主な対象・内容 |
|--------|--------|---------------|
| 生活保護法 | 1946年（旧法）、1950年（新法） | 最低生活の保障と自立助長 |
| 児童福祉法 | 1947年 | 障がい児を含む児童の福祉 |
| 身体障害者福祉法 | 1949年 | 身体障がい者の更生援護 |

身体障害者福祉法は、日本で最初の障がい者を対象とした専門的な福祉法です。しかし、この時代の制度は「戦傷病者の社会復帰」という文脈が色濃く、対象は主に身体障がいに限定されていました。

### 1.3 この時代の障害観

この時代の法制度は、障害の医学モデルに完全に依拠していました。障害は「個人の心身の異常」であり、支援とは「治療・訓練・更生」を意味しました。「更生」という言葉が法律名にも登場するように、障がい者は「元の状態に戻るべき人」として位置づけられていました。

## 第2章：福祉六法体制と施設福祉の拡大（1960〜1980年代）

### 2.1 精神薄弱者福祉法と精神衛生法

| 法律名 | 制定年 | 主な内容 |
|--------|--------|---------|
| 精神衛生法 | 1950年 | 精神障がい者の医療および保護（主に入院治療中心） |
| 精神薄弱者福祉法 | 1960年 | 知的障がい者の福祉サービス（後の「知的障害者福祉法」） |

この時期、精神障がい者に対しては「医療モデル」が支配的で、精神科病院への長期入院（社会的入院）が常態化しました。知的障がい者に対しても、大規模入所施設での生活が主流となります。

### 2.2 心身障害者対策基本法の制定（1970年）

1970年、障がい者施策全般の基本理念を定める「心身障害者対策基本法」が制定されました。これは後の「障害者基本法」の前身です。

しかし、この法律の名称自体が「対策」という言葉を含んでおり、障がい者は「対策の対象」＝「管理すべき存在」として位置づけられていました。

### 2.3 措置制度の特徴

この時代の障がい福祉は「措置制度」によって運営されていました。

**措置制度の仕組み**

- 行政（市町村・都道府県）が、障がい者に対するサービスの種類・量・利用先を決定する
- 本人に選択権はない（行政処分としてのサービス提供）
- 費用は公費負担（応能負担）
- サービス提供者は行政の委託先であり、利用者と直接の契約関係にはない

この制度の下では、障がい者は「サービスの受け手」に過ぎず、自ら支援を選択する主体としては位置づけられていませんでした。

### 2.4 国際障害者年と権利意識の萌芽

1981年、国連の「国際障害者年」が「完全参加と平等」をテーマに展開されました。1983年からの「国連障害者の十年」とあわせて、日本でも障がい者の権利に関する議論が始まります。

## 第3章：パラダイムシフトの胎動（1990年代）

### 3.1 障害者基本法への改称（1993年）

1993年、「心身障害者対策基本法」が全面改正され、「障害者基本法」に改称されました。

**主な改正点**

- 「心身障害者」から「障害者」へ（対象の拡大を示唆）
- 精神障がい者が障害者基本法の対象に明確に位置づけられた
- 障害者基本計画の策定が義務化
- 「対策」から「施策」へ（微妙だが重要な言葉の変化）

### 3.2 精神保健福祉法の成立（1995年）

精神衛生法は1987年に「精神保健法」に改称された後、1995年に「精神保健及び精神障害者福祉に関する法律（精神保健福祉法）」へと改正されました。

この改正は、精神障がい者に対する施策が「医療中心」から「医療＋福祉」へと転換する画期的な一歩でした。精神障害者保健福祉手帳制度もこの時に創設されています。

### 3.3 社会福祉基礎構造改革（1990年代後半）

1990年代後半、社会福祉全体にわたる大規模な構造改革が議論されました。その核心は、措置制度から契約制度への転換です。

**社会福祉基礎構造改革の理念**

- 利用者の自己決定と選択の尊重
- サービスの質の向上と競争の導入
- 情報公開と透明性の確保
- 措置（行政処分）から契約（対等な関係）へ

この改革は2000年の介護保険制度の導入で先行的に実現し、障がい福祉にも波及していきます。

## 第4章：支援費制度 ― 契約制度への転換（2003年）

### 4.1 支援費制度の導入

2003年、障がい福祉サービスの提供方法が、措置制度から支援費制度へと転換しました。

**支援費制度の特徴**

| 項目 | 措置制度 | 支援費制度 |
|------|---------|-----------|
| サービスの決定者 | 行政 | 利用者本人 |
| 利用者の位置づけ | 措置の客体 | 契約の主体 |
| 事業者との関係 | 行政の委託先 | 利用者との直接契約 |
| 費用負担 | 応能負担 | 応能負担（利用契約に基づく） |

これは障がい者が「サービスを選ぶ主体」として位置づけられた、歴史的な転換でした。

### 4.2 支援費制度の課題

しかし、支援費制度は開始直後から深刻な財政的課題に直面しました。

- サービス利用の急増による財源不足（特にホームヘルプサービス）
- 精神障がい者が制度の対象外であった
- 地域間格差（サービス基盤の整備状況の差）
- 障害種別（身体・知的・精神）ごとの制度の分立

この財政的な行き詰まりが、次の大きな制度改革 ― 障害者自立支援法の成立 ― へとつながります。

## 第5章：障害者自立支援法 ― 統合と応益負担の衝撃（2005年）

### 5.1 障害者自立支援法の成立

2005年に成立し2006年4月から施行された障害者自立支援法は、日本の障がい福祉制度を根本的に再編した法律です。

**障害者自立支援法の主な特徴**

- **三障がいの一元化**: 身体障がい・知的障がい・精神障がいのサービスを一つの法律で一元的に提供
- **応益負担（定率負担）の導入**: サービス利用量に応じた原則1割の自己負担
- **障害程度区分の導入**: 全国統一の基準によるサービス支給量の決定
- **就労支援の体系化**: 就労移行支援、就労継続支援A型・B型の創設
- **地域生活支援事業の創設**: 市町村による柔軟なサービス提供

### 5.2 就労支援体系の整備

障害者自立支援法は、それまでバラバラだった就労支援サービスを体系化しました。

| サービス類型 | 内容 | 特徴 |
|-------------|------|------|
| 就労移行支援 | 一般就労に向けた訓練・支援 | 原則2年間の利用期限 |
| 就労継続支援A型 | 雇用契約に基づく就労機会の提供 | 最低賃金が保障される |
| 就労継続支援B型 | 雇用契約を結ばない就労機会の提供 | 利用期限なし、工賃は低水準 |

これにより、旧来の「授産施設」「福祉工場」「小規模作業所」等が新体系に移行していきました。

### 5.3 応益負担をめぐる激しい反発

障害者自立支援法に対しては、特に応益負担（定率1割負担）の導入をめぐって、障がい当事者団体から激しい反対運動が起こりました。

「障害が重いほどサービスを多く使う → 障害が重いほど自己負担が大きくなる」という構造は、「障害は自己責任なのか」という根本的な問いを突きつけました。

2008年以降、特別対策として負担軽減措置が相次いで導入され、2010年には応益負担を応能負担に改める法改正が行われました。

### 5.4 違憲訴訟と基本合意

応益負担に対しては全国各地で違憲訴訟が提起され、2010年1月、国（厚生労働省）と原告団との間で「基本合意文書」が締結されました。この合意では、国が障害者自立支援法の制度的問題を認め、新たな法制度の構築を約束しました。

## 第6章：障害者権利条約の批准に向けた国内法整備（2011〜2014年）

### 6.1 条約批准に向けたロードマップ

日本政府は2007年に障害者権利条約（CRPD）に署名しましたが、国内法の整備が不十分であるとして批准を留保しました。以下の国内法整備を経て、2014年に批准に至ります。

| 年 | 法整備 | 意義 |
|----|--------|------|
| 2011年 | 障害者基本法の改正 | 「社会的障壁」の概念を導入、障害の定義を社会モデルに接近させた |
| 2012年 | 障害者総合支援法の成立 | 障害者自立支援法を改称・改正。難病等を対象に追加 |
| 2013年 | 障害者差別解消法の制定 | 不当な差別的取扱いの禁止、合理的配慮の提供 |
| 2013年 | 障害者雇用促進法の改正 | 雇用における差別禁止・合理的配慮の義務化 |
| 2014年1月 | 障害者権利条約の批准 | 条約の国内的効力の発生 |

### 6.2 障害者基本法の改正（2011年）

2011年の改正は、日本の障害法制において最も重要な転換点の一つです。

**障害の定義の変更**

- **改正前**: 「身体障害、知的障害又は精神障害があるため、継続的に日常生活又は社会生活に相当な制限を受ける者」
- **改正後**: 「身体障害、知的障害、精神障害（発達障害を含む。）その他の心身の機能の障害がある者であつて、障害及び社会的障壁により継続的に日常生活又は社会生活に相当な制限を受ける状態にあるもの」

この改正により、①発達障がいが明記され、②「社会的障壁」が障害の原因として認められました。これは、ICIDHの医学モデルからICFの相互作用モデル、さらにはCRPDの社会モデルへの制度的な転換を意味しています。

### 6.3 障害者総合支援法の成立（2012年）

障害者自立支援法は、その名称から「自立」という言葉が削除され、「障害者の日常生活及び社会生活を総合的に支援するための法律（障害者総合支援法）」に改められました。

**主な変更点**

- 法の目的に「基本的人権を享有する個人としての尊厳」を明記
- 「障害程度区分」を「障害支援区分」に変更（障がいの程度ではなく、必要な支援の度合いを測る）
- 難病等の患者を障がい福祉サービスの対象に追加
- 重度訪問介護の対象拡大

### 6.4 障害者差別解消法の制定（2013年）

2013年に制定された障害者差別解消法は、CRPDの国内実施法としての性格を持ちます。

**基本的な枠組み**

- **不当な差別的取扱いの禁止**: 行政機関・民間事業者ともに法的義務
- **合理的配慮の提供**: 行政機関は法的義務、民間事業者は努力義務（制定当初）

### 6.5 障害者雇用促進法の改正（2013年）

2013年の改正で、障害者雇用促進法にも差別禁止と合理的配慮の提供義務が組み込まれました。

## 第7章：近年の法改正と現在の制度（2018年〜2026年）

### 7.1 障害者総合支援法の段階的改正

| 年 | 改正内容 |
|----|---------|
| 2018年 | 就労定着支援の創設（就職後最大3年間のフォローアップ）、自立生活援助の創設 |
| 2022年改正（2024年施行） | 就労選択支援の創設（2025年10月施行予定）、グループホームの一人暮らし等への支援 |

### 7.2 就労支援サービスの現在の体系

2025年度以降の就労支援サービスは、以下の5類型で構成されます。

| サービス | 概要 | 対象 | 特徴 |
|---------|------|------|------|
| 就労選択支援 | 就労先・働き方の選択を支援 | 就労を希望する障がい者 | 2025年10月〜新設。本人の意思決定を中心に据えた支援 |
| 就労移行支援 | 一般就労に向けた訓練・支援 | 一般就労を希望する65歳未満の障がい者 | 原則2年。就労定着率による報酬体系 |
| 就労継続支援A型 | 雇用契約に基づく就労の機会提供 | 一般企業への就職が困難な障がい者 | 最低賃金保障。スコア方式による報酬評価 |
| 就労継続支援B型 | 雇用契約を結ばない就労の機会提供 | A型・一般就労が困難な障がい者 | 平均工賃月額に連動した報酬体系 |
| 就労定着支援 | 就職後の職場定着を支援 | 一般就労に移行した障がい者 | 最大3年間。定着率に連動した報酬体系 |

### 7.3 障害者差別解消法の改正（2024年4月施行）

2021年の改正（2024年4月施行）により、民間事業者における合理的配慮の提供が法的義務に格上げされました。

これは就労支援事業所にも直接適用される極めて重要な改正です。就労支援事業所は、障がい福祉サービスの提供者であると同時に、障がい者に就労の場を提供する「事業主」としての性格も持ちます。合理的配慮の提供義務化は、事業所運営の根幹に関わります。

### 7.4 障害者雇用促進法の段階的引き上げ

法定雇用率は段階的に引き上げられています。

| 時期 | 法定雇用率 | 対象企業 |
|------|-----------|---------|
| 2024年4月〜 | 2.5% | 従業員40人以上 |
| 2026年7月〜 | 2.7% | 従業員37.5人以上 |

また、2024年から週10時間以上20時間未満の短時間労働者も算定対象に加わるなど、「多様な働き方」を反映した制度改正が進んでいます。

### 7.5 令和6年度障害福祉サービス報酬改定（2024年度）

報酬改定においても、アウトカム（成果）志向が強まっています。

就労継続支援B型の報酬体系は平均工賃月額に直接連動しており、定員20人以下の事業所で工賃月額45,000円以上を達成すれば837単位、一方でベースラインの45単位との間に大きな格差が存在します。

## 第8章：2022年 CRPD対日審査・総括所見 ― 国際的な評価

### 8.1 初回審査の概要

2022年8月、スイス・ジュネーブにおいて、障害者権利委員会による日本の初回審査が実施されました。同年9月に発出された総括所見では、93項目の懸念と92項目の勧告が示されました。

### 8.2 就労に関する主要な勧告

**委員会の懸念（パラグラフ57）**

- 多くの障がい者がシェルタード・ワークショップ（就労継続支援B型等）に留まり、開かれた労働市場への移行が進んでいない
- 作業所での工賃が極めて低く、生計を立てるのに十分でない

**委員会の勧告（パラグラフ58）**

- 作業所から開かれた労働市場への移行の迅速化
- 同一価値の労働についての同一報酬の保障
- 職場のアクセシビリティと合理的配慮の確保
- 知的障がい者・精神障がい者・障がいのある女性の雇用促進

### 8.3 その他の重要な勧告

| 条文 | 勧告の要点 |
|------|-----------|
| 第12条（法的能力） | 成年後見制度（代行決定）から支援付き意思決定への転換 |
| 第19条（脱施設化） | 施設収容の廃止、予算を地域生活支援に振り向ける |
| 第24条（教育） | 分離教育の中止、インクルーシブ教育への国家行動計画策定 |
| 第14条（身体の自由） | 精神科への強制入院を可能にしている法律の廃止 |

これらの勧告は、日本が制度的にもまだ医学モデルから十分に脱却できていないことを国際社会が指摘したものとして受け止める必要があります。

## 第9章：年表 ― 日本の障がい福祉法制度の変遷

| 年 | 出来事 | 意義 |
|----|--------|------|
| 1946年 | 日本国憲法公布（第25条：生存権） | 障がい福祉法制度の憲法的基盤 |
| 1947年 | 児童福祉法制定 | 障がい児を含む児童福祉の基盤 |
| 1949年 | 身体障害者福祉法制定 | 日本初の障がい者専門福祉法 |
| 1950年 | 精神衛生法制定 | 精神障がい者の医療・保護（入院中心） |
| 1960年 | 精神薄弱者福祉法制定 | 知的障がい者福祉の法的基盤 |
| 1970年 | 心身障害者対策基本法制定 | 障がい者施策の基本法（ただし「対策」の客体） |

## 第10章：変遷の構造的理解 ― 4つの転換軸

日本の障がい福祉法制度の変遷は、以下の4つの軸で整理できます。

### 転換軸1：障害観の転換

```
医学モデル（個人の問題）
  → 社会モデル（社会の問題）
    → 相互作用モデル/ICF（個人と環境の動的関係）
      → 人権モデル/CRPD（権利の主体としての障がい者）
```

### 転換軸2：サービス提供の転換

```
措置制度（行政が決定）
  → 支援費制度（利用者が選択・契約）
    → 障害者自立支援法（三障がい一元化・応益負担→応能負担に修正）
      → 障害者総合支援法（総合的支援・就労選択支援の創設）
```

### 転換軸3：生活の場の転換

```
施設収容（大規模入所施設・精神科病院の長期入院）
  → 地域生活への移行（グループホーム・居宅サービス）
    → 地域での自立した生活（本人の選択に基づく居住形態）
```

### 転換軸4：就労支援の転換

```
保護的就労（授産施設・福祉工場）
  → 体系化された就労支援（移行・A型・B型）
    → 一般就労への移行重視＋定着支援
      → 本人の意思に基づく就労選択支援
```

## 第11章：残された課題と今後の展望

### 11.1 B型事業所の工賃問題

令和5年度（2023年度）のB型全国平均工賃月額は約17,031円です。CRPDが求める「同一価値の労働についての同一報酬」（第27条）との乖離は依然として大きく、構造的な解決が求められています。

### 11.2 精神科長期入院の問題

2022年時点で約7万人が精神科病院に1年以上入院しています。CRPDの総括所見では、精神科への強制入院を可能にする法律の廃止が勧告されており、日本は重大な制度的課題を抱えています。

### 11.3 成年後見制度と意思決定支援

CRPDは「代行決定」から「支援付き意思決定」への転換を求めていますが、日本の成年後見制度はなお代行決定モデルに基づいています。就労支援の現場でも、「本人の意思と選好」に基づく支援の実践が課題です。

### 11.4 障害者手帳制度の限界

現行の障害者手帳制度は医学モデルに基づく障害認定であり、CRPDが求める社会モデルに基づくニーズ判定への転換が勧告されています。手帳がなければサービスを受けられないという制度的障壁は、「制度の谷間」の問題を生んでいます。

## おわりに：法制度の変遷が支援者に問いかけるもの

法律は変わりました。しかし、現場の支援は変わっているでしょうか。

措置制度は廃止されましたが、支援者が利用者の代わりにサービスを決めていないでしょうか。障害者基本法は「社会的障壁」を認めましたが、アセスメントは依然として「本人のできないこと」のリストになっていないでしょうか。合理的配慮は義務化されましたが、「前例がない」「予算がない」を理由に拒んでいないでしょうか。

法制度の変遷は、「障害とは何か」「支援とは何か」という問いに対する社会の答えが変わってきた歴史です。そして、その答えはまだ完成していません。

次に変わるべきは、私たちの支援のあり方です。
