## はじめに：なぜパターナリズムを学ぶのか

障害福祉の現場で、「この人のためになることを」「本人の最善の利益を考えて」と言うことがあります。一見正しい姿勢に思えるこの言葉が、実は障害のある人の権利を奪う道具になり得る。この構造を「パターナリズム（父権主義）」と呼びます。

パターナリズムを理解することは、支援者としてのプロフェッショナリズムの核心です。なぜなら、善意と虐待の境界線はこの構造の中にあるからです。

---

## 第1部：パターナリズムとは何か

### 1.1 定義

パターナリズム（paternalism）とは「あなたのためを思って、あなたに代わって決めてあげる」という態度です。語源はラテン語の「pater（父）」に由来し、父親が子どものために判断を代行するという家族的比喩から来ています。

一見すると優しさに見えますが、その本質は**「あなたには自分で決める能力がない」という前提**に立っています。

### 1.2 パターナリズムの3つの構造的問題

**問題1：誰が「最善」を決めるのか**

「最善の利益」を判断するのは本人ではなく、第三者（後見人、医師、支援者、家族）です。しかし、何が「最善」かは本人の価値観、人生経験、希望によって異なります。第三者が判断する「最善」は、多くの場合、**第三者自身の価値観を反映したもの**になりがちです。

**問題2：「能力がない」という前提**

「最善の利益」基準を適用するということは、「この人には自分で判断する能力がない」と認定することを意味します。しかし、意思決定能力は固定的なものではなく、**適切な支援があれば多くの場合行使可能**なものです。「能力がない」と判断してしまうことで、能力を発揮する機会そのものが奪われるという悪循環が生じます。

**問題3：権力の非対称性**

「最善の利益」を判断する側と判断される側の間には、圧倒的な権力の差があります。判断する側には専門知識、制度的権限、情報へのアクセスがあり、判断される側にはそれらがない。この非対称性のもとでは、「最善の利益」は事実上**「判断する側の都合」**になりやすいのです。

### 1.3 ベネボレント・パターナリズム（温情的パターナリズム）

特に福祉の現場で多いのが「ベネボレント（温情的）パターナリズム」です。

> 「あなたのためを思って」という善意に基づき、家長（あるいは支援者・企業・国家）が個人の決定権に介入する。これは安心感を生むが、個人の自律性（Autonomy）を損なう。

善意に基づいているからこそ、本人も周囲も問題だと気づきにくい。これがベネボレント・パターナリズムの最大の危険性です。

## 第2部：日本におけるパターナリズムの歴史的展開

### 2.1 措置制度（〜2000年）：パターナリズムの制度化

日本の障害福祉制度は、長らく「措置制度」というパターナリズムの極致にありました。

| 項目 | 措置制度の特徴 |
|------|-------------|
| **時期** | 〜2000年 |
| **構造** | 行政処分による一方的決定 |
| **関係性** | 行政 → 施設 → 利用者（垂直的） |
| **責任** | 行政が全責任を負う |
| **問題** | 利用者の選択権なし ＝ パターナリズム |

行政が「あなたにはこの施設が必要です」と判断し、利用者に選択肢はなかった。支援は「してあげる」ものであり、利用者は「される」側でしかなかった。

### 2.2 支援費制度（2003年〜）：パターナリズムからの脱却の始まり

2003年の支援費制度は、措置制度を根本から変えました。

| 項目 | 支援費制度の特徴 |
|------|---------------|
| **構造** | 利用者と事業者の契約 |
| **関係性** | 利用者 ↔ 事業者（対等な契約関係） |
| **選択権** | 利用者が自分でサービスを選べる |
| **意義** | 初めて「利用者本位」が制度化 |

**措置から選択へ** ― これはパターナリズムからの制度的脱却の第一歩でした。

### 2.3 成年後見制度：残存するパターナリズム

しかし、成年後見制度には依然としてパターナリズム的構造が残っています。

- 後見人が「最善の利益」を名目に、本人の法的行為能力を制限
- 後見類型では本人の意思が事実上無視されうる
- 2026年の制度改革で、後見・保佐の類型を廃止し補助に一元化する方向性が議論中

### 2.4 日本的「家」システムとパターナリズム

日本には「家（イエ）」制度という独自のパターナリズム構造が存在します。

**包摂と抑圧の二面性**:
- 「家」は構成員に対し、終身雇用的な「生存保障」を提供する
- 一方で、個人の自由や物語を「家」の論理に従属させることを要求する
- 「滅私奉公」に象徴される ― 大きな物語（トランスナラティブ）が強大すぎると、個人の小さな物語（Personal Narrative）は圧殺される

福祉の現場でも同様の構造が見られます。支援者が利用者の「すべて（生い立ち、障害、生活歴）」を把握しようとするパターナリズムは、利用者を「管理される客体」に固定化します。

## 第3部：CRPD（障害者権利条約）によるパラダイムシフト

### 3.1 CRPD第12条の革命

障害者権利条約（CRPD）第12条は、パターナリズムに対する決定的な回答を示しました。

> 締約国は、障害者が生活のあらゆる側面において他の者と平等に法的能力を享有することを認める。

3つのポイント:
1. **法的能力の平等**: 障害を理由に法的行為能力を否定・制限することは差別
2. **支援の提供**: 能力行使に支援が必要な場合、その支援を提供する義務
3. **セーフガード**: 支援が本人の権利を侵害しないための保護措置

### 3.2 一般的意見第1号（2014年）

CRPD委員会の一般的意見第1号は、この問題に歴史的転換をもたらしました。

> **成人の障害者に関して「最善の利益」原則を適用することは、第12条に基づく支援付き意思決定の原則と両立しない。**

### 3.3 代行決定から支援付き意思決定へ

| 従来（パターナリズム） | CRPDが求める姿 |
|----------------------|---------------|
| 最善の利益（Best Interest） | **本人の意思と選好（Will and Preferences）** |
| 第三者が「何が良いか」を判断 | 本人が「何を望むか」を表明し、それを支援 |
| 客観的な「正解」があるという前提 | 本人の主観的な価値観が最優先 |
| 判断能力の有無で適用を決める | 判断能力は支援によって行使可能 |
| **代行決定**（Substituted Decision-Making） | **支援付き意思決定**（Supported Decision-Making） |

## 第4部：福祉の現場に潜むパターナリズム

### 4.1 グレーゾーンの具体例

日常の支援に潜むパターナリズムの具体例:

| 行為 | 表面的な理由 | パターナリズムの本質 |
|------|-------------|-------------------|
| トイレに行ける方におむつを使う | 失敗させないため | スタッフの都合の優先 |
| 自分で飲める薬を食事に混ぜる | 飲み忘れ防止 | 自己決定権の剥奪 |
| 自分で食べられるのに全部介助する | 効率化 | 自立心の破壊 |
| 大人を本人の同意なくあだ名で呼ぶ | 親しみ | 人格の尊重の欠如 |
| 「この人にはこれが一番いい」と決める | 専門的判断 | 本人の声を聞かない |
| 「安全のため」の不必要な行動制限 | リスク管理 | 過度な保護による自由の制限 |

### 4.2 「よかれと思って」のメカニズム

善意が虐待に転化するメカニズム:

1. 支援者が「この人のために」と考える（善意）
2. 本人の意思を確認せずに判断を代行する（パターナリズム）
3. 利用者が「自分でできること」を奪われる（能力の剥奪）
4. 自立心が削がれ、依存が深まる（学習性無力感）
5. 「やっぱりこの人には支援が必要」と確認バイアスが強化される
6. さらなるパターナリズムの正当化（悪循環の完成）

### 4.3 虐待統計が示すパターナリズムの帰結

令和6年度の虐待統計データは、パターナリズムの構造的帰結を実証しています。

- 施設従事者による虐待の発生要因:
  - **教育・知識・介護技術等に関する問題: 67.5%**
  - **倫理観や理念の欠如: 60.2%**
  - **職員のストレス・感情コントロールの問題: 58.7%**
- 養護者による虐待で**虐待者が虐待と認識していない: 40.1%**

善意に基づくパターナリズムが、自覚なき虐待を生み出している構造が統計的に裏付けられています。

### 4.4 適応的選好（Adaptive Preferences）の問題

哲学者ロベインズ（Robeyns）が指摘する「適応的選好」は、パターナリズムの環境下で深刻化します。

> **適応的選好とは**: 抑圧された人々が低い期待に適応してしまう問題

- 長期間パターナリズム的支援を受けた利用者は、「自分には決められない」「支援者に任せた方がいい」と学習する
- これは本人の「真の選好」ではなく、パターナリズム的環境への適応
- SoEフレームワークでは「18歳の夢メソッド」（適応前の真の選好発掘）で対抗

## 第5部：パターナリズムを超えるために

### 5.1 制度的転換のまとめ

| 転換前（パターナリズム） | 転換後（権利ベース） | 制度的根拠 |
|------------------------|-------------------|-----------|
| 医学モデル | **社会モデル** | CRPD前文(e) |
| 措置制度 | **契約制度** | 支援費制度（2003年） |
| 代行決定 | **支援付き意思決定** | CRPD第12条 |
| 最善の利益 | **本人の意思と選好** | 一般的意見第1号 |
| 保護の対象 | **権利の主体** | 障害者基本法改正（2011年） |
| 障害程度区分 | **障害支援区分** | 総合支援法（2012年） |

### 5.2 支援者への実践的指針

パターナリズムに陥らないための日常的チェック:

1. **「この人のため」と思った時に立ち止まる**: それは本当に本人が望んでいることか？
2. **本人に聞く**: 判断を代行する前に、必ず本人の意思を確認する
3. **選択肢を提示する**: 「これにしましょう」ではなく「AとBがありますが、どちらがいいですか？」
4. **できることを奪わない**: 時間がかかっても、本人ができることは本人にやってもらう
5. **「安全」を理由にした過度な制限を疑う**: リスクゼロは不可能。本人の自由とのバランスを考える
6. **チームで振り返る**: 一人の判断に頼らず、複数の視点で支援を検証する

### 5.3 パターナリズムから脱却した支援の姿

> 支援者の役割は「あなたに代わって決めること」ではなく、「あなたが自分で決められるように支えること」。

これが「支援付き意思決定」の本質であり、CRPDが求めるパラダイムシフトの中核です。

## 第6部：SoE理論・Input Constitutional AI との接続

### 6.1 トバシ（簿外処理）としてのパターナリズム

SoEフレームワークの会計メタファーで言えば、パターナリズムによる代行決定は「トバシ（簿外処理）」として機能します。

- 本人の意思（資産）を帳簿から消去する
- 代わりに第三者の判断を「本人の利益」として計上する
- 本人の自己決定権という最も重要な資産が、見えないところで毀損されていく

### 6.2 複式簿記メタファーによる対抗

パターナリズムを構造的に防ぐための装置:

- **バランスシート（BS）**: 利用者の累積環境因子（過去の負債や資産）を可視化
- **損益計算書（PL）**: 現在の支援活動の効果を追跡
- **情報境界線**: 本人が主体的に自己理解を行い、支援者には「合理的配慮の根拠」としてのみ提示

これにより、パターナリズム（過干渉）を防ぎつつ、適切なサポートを受けるという自律的な関係が構築可能になります。

### 6.3 Input Constitutional AI

記録作成段階での権利保障 ― 支援記録にパターナリズム的判断が混入していないかを、AIが構造的に検出する仕組み。

- 「本人の意思」と「支援者の判断」が記録上で明確に区別されているか
- 「最善の利益」の名のもとに本人の選好が上書きされていないか
- 支援計画が本人の意思と選好に基づいているか

### 6.4 三因子モデルとの接続

| 因子 | パターナリズムとの関係 |
|------|---------------------|
| **Hubris Factor（傲慢因子）** | 「自分は正しい判断ができる」という支援者の特権意識がパターナリズムを駆動 |
| **Anomie Factor（アノミー因子）** | パターナリズムによって自己決定権を奪われた利用者が規範を喪失する |
| **Hypnos Factor（催眠因子）** | 「障害者は保護の対象」という社会全体のロール固定化がパターナリズムを維持 |
