# 障害者権利条約（CRPD）— 就労支援事業所スタッフ向け教育資料

## この文書の目的

本文書は、就労支援事業所（就労継続支援A型・B型、就労移行支援等）のスタッフが、障害者権利条約（CRPD: Convention on the Rights of Persons with Disabilities）の基本理念と実務への影響を理解するための教育資料です。NotebookLMでの音声教材生成を前提に構成しています。

---

## 第1章 障害者権利条約とは何か

### 1.1 条約の概要

障害者権利条約（CRPD）は、2006年12月13日に国連総会で採択され、2008年5月3日に発効した国際条約です。障害者の人権と基本的自由の享有を確保し、障害者の固有の尊厳の尊重を促進するための包括的な人権条約です。

- **採択**: 2006年12月13日（国連総会）
- **発効**: 2008年5月3日
- **日本の署名**: 2007年9月28日
- **日本の批准**: 2014年1月20日
- **締約国数**: 190か国以上（2025年時点）

### 1.2 なぜこの条約が重要か

障害者権利条約は、障害者に新しい権利を与えるものではなく、**既に存在する人権を障害者が実質的に享有できるようにする**ための条約です。つまり「障害者にも人権がある」のではなく「人権はもともと全員にある。それが障害者には届いていなかった」という認識に立っています。

### 1.3 条約の構成

- **前文（Preamble）**: 条約の理念と背景
- **第1条〜第50条**: 実体規定と手続規定
- **選択議定書**: 個人通報制度（日本は未批准）

## 第2章 「障害の社会モデル」— 最も重要なパラダイム転換

### 2.1 医学モデルと社会モデル

障害者権利条約の根幹にあるのは、**障害の社会モデル**（Social Model of Disability）という考え方です。

| | 医学モデル（旧来の考え方） | 社会モデル（条約の考え方） |
|---|---|---|
| **障害の原因** | 個人の心身の機能障害（インペアメント） | 社会の側にある障壁（バリア） |
| **問題の所在** | 本人の身体・精神 | 社会の制度・環境・意識 |
| **解決方法** | 治療・リハビリ・本人の努力 | 社会の障壁の除去・環境整備 |
| **具体例** | 「車椅子の人は階段を上れない」 | 「階段しかない建物が問題」 |
| **支援の方向** | 本人を"普通"に近づける | 社会を変える |

### 2.2 就労支援現場での社会モデルの意味

従来の就労支援は、利用者の「できないこと」を訓練で克服させるアプローチ（医学モデル的）が主流でした。社会モデルの視点では、以下のように発想が転換されます：

- **従来**: 「この利用者はコミュニケーションが苦手だから訓練が必要」
- **社会モデル**: 「この職場のコミュニケーション方法が合っていないのでは？」

- **従来**: 「集中力が続かないから短時間しか働けない」
- **社会モデル**: 「作業環境の刺激（音・光・人の動き）を調整できないか？」

- **従来**: 「一般就労は難しいのでB型で」
- **社会モデル**: 「一般就労を阻んでいる社会的障壁は何か？」

### 2.3 インペアメント（機能障害）とディスアビリティ（障害）の区別

社会モデルでは、以下を明確に区別します：

- **インペアメント（impairment）**: 個人の身体的・精神的・知的な機能の違い
- **ディスアビリティ（disability）**: インペアメントと社会的障壁の相互作用によって生じる制約

例：聴覚障害（インペアメント）のある人が、手話通訳のない会議に参加できない（ディスアビリティ）。問題は聴覚障害ではなく、手話通訳が用意されていないこと。

## 第3章 条約の一般原則（第3条）

条約の第3条は、以下の8つの一般原則を定めています。これらは条約全体を貫く基本的な価値観です。

1. **固有の尊厳の尊重、個人の自律（自ら選択する自由を含む）及び個人の自立の尊重**
2. **非差別**
3. **社会への完全かつ効果的な参加及びインクルージョン（包容）**
4. **差異の尊重と、人間の多様性の一部かつ人類の一員としての障害者の受容**
5. **機会の均等**
6. **アクセシビリティ（施設及びサービス等の利用の容易さ）**
7. **男女の平等**
8. **障害のある児童の発達しつつある能力の尊重と、アイデンティティを保持する権利の尊重**

### 就労支援への適用

| 原則 | 就労支援での意味 |
|---|---|
| 個人の自律 | 利用者自身が働き方を選択する権利がある |
| 非差別 | 障害を理由に就労の機会を制限してはならない |
| インクルージョン | 分離された場ではなく、社会の中での就労を目指す |
| 差異の尊重 | 「普通の働き方」に合わせるのではなく、多様な働き方を認める |
| 機会の均等 | スタートラインを揃える（合理的配慮の提供） |
| アクセシビリティ | 物理的・情報的バリアの除去 |

## 第4章 就労支援に直結する条文

### 4.1 第27条 労働及び雇用

条約の中で就労支援に最も直接的に関わるのが第27条です。

> 締約国は、障害者が他の者との平等を基礎として労働についての権利を有することを認める。この権利には、障害者にとって開かれ、インクルーシブ（包容的）で、かつ、アクセシブル（利用しやすい）な労働市場及び労働環境において、障害者が自由に選択し又は引き受けた労働を通じて生計を立てる機会を得る権利を含む。

**第27条が締約国に求めること（主要項目）**:

- (a) あらゆる形態の雇用に係るすべての事項における障害に基づく差別の禁止
- (b) 他の者との平等を基礎とする、公正かつ良好な労働条件の保護（同一価値の労働についての同一報酬を含む）
- (d) 障害者が一般的な技術及び職業の指導プログラム、職業紹介サービス並びに職業訓練及び継続的な訓練を効果的に利用する機会を得ること
- (e) 労働市場における障害者の雇用機会の増大及び昇進の促進
- (g) 公的部門における障害者の雇用
- (h) 適当な政策及び措置（積極的差別是正措置を含み得る）を通じた民間部門における障害者の雇用の促進
- (i) 職場における合理的配慮の提供の確保
- (j) 開かれた労働市場における障害者の職業経験の取得の促進
- (k) 障害者のための職業リハビリテーション、職業の保持及び職場復帰の計画の促進

### 4.2 第12条 法律の前にひとしく認められる権利（法的能力）

就労支援の現場で見落とされがちだが極めて重要な条文です。

> 締約国は、障害者がすべての場所において法律の前に人として認められる権利を有することを再確認する。

**核心**: 障害者は、障害のない人と平等に法的能力を有する。成年後見制度等の「代行決定」（代わりに決める）ではなく、**「支援付き意思決定」**（本人が決めるのを支援する）への転換が求められている。

**就労支援での意味**:
- 「この利用者にはB型が適切」と支援者が決めるのではなく、利用者本人が情報を得た上で選択する
- 意思決定が難しい場合でも、「代わりに決める」のではなく「決められるように支援する」
- 契約、金銭管理、就職活動における本人の意思を最大限尊重する

### 4.3 第19条 自立した生活及び地域社会への包容

> 締約国は、すべての障害者が他の者と平等の選択の機会をもって地域社会で生活する平等の権利を有することを認める。

**就労支援での意味**:
- 就労は「施設内の作業」で完結するのではなく、地域社会での自立した生活の一部として位置づけられる
- 入所施設からの地域移行と就労支援は一体的に考える必要がある
- グループホームから通所する利用者についても、本人の選択による居住形態であるか確認する

### 4.4 第5条 平等及び非差別

> 締約国は、障害に基づくあらゆる差別を禁止するものとし、いかなる理由による差別に対しても平等かつ効果的な法的保護を障害者に保障する。

**合理的配慮の否定は差別である**:
条約は「合理的配慮の否定」を差別の一形態として明確に定義しています。つまり、合理的配慮を提供しないことは、単なる「サービスの不足」ではなく「差別」です。

## 第5章 2022年 対日審査・総括所見 — 日本への勧告

### 5.1 審査の経緯

2022年8月22日〜23日、スイス・ジュネーブにおいて、障害者権利委員会による日本の初回審査（建設的対話）が実施されました。9月9日に総括所見（Concluding Observations）が採択・公表されました。

- **懸念**: 93項目
- **勧告**: 92項目
- **次回報告期限**: 2028年2月20日

### 5.2 就労に関する勧告（パラグラフ57-58）

**委員会の懸念（パラ57）**:
- 障害者の多くがシェルタード・ワークショップ（作業所 = 就労継続支援B型等）に留まり、開かれた労働市場への移行が進んでいない
- 作業所での工賃が極めて低く、生計を立てるのに十分でない

**委員会の勧告（パラ58）**:

1. **作業所から開かれた労働市場への移行の迅速化**
   - シェルタード・ワークショップ及び雇用に関連した福祉サービスから、民間及び公的部門における開かれた労働市場への障害者の移行努力を強化すること
   - 包容的な労働環境において、同一価値の労働についての同一報酬を保障すること

2. **職場のアクセシビリティと合理的配慮**
   - 職場の建物環境が障害者にとって利用しやすく調整されたものであることを確保する
   - 個別的な支援及び合理的配慮に関する訓練をあらゆる段階の雇用者に提供する

3. **特に脆弱な立場にある障害者の雇用促進**
   - 知的障害者、精神障害者、障害のある女性の公的・民間部門における雇用を特に促進する

### 5.3 その他の重要な勧告

| 条文 | 勧告の要点 |
|------|-----------|
| 第12条（法的能力） | 成年後見制度（代行決定）から支援付き意思決定への転換 |
| 第19条（脱施設化） | 施設収容の廃止、予算を地域生活支援に振り向ける |
| 第24条（教育） | 分離教育の中止、インクルーシブ教育への国家行動計画策定 |
| 第14条（身体の自由） | 精神科への強制入院を可能にしている法律の廃止 |

### 5.4 「Nothing About Us, Without Us」の原則

総括所見全体を通じて、日本政府に対し**障害者団体との緊密な協議**が繰り返し求められています。「私たち抜きに私たちのことを決めないで」（Nothing About Us, Without Us）は、条約の根幹にある理念です。

就労支援の現場でも同様です。**支援計画は利用者と共に作る**。利用者の「声」が個別支援計画に反映されているか、形式的な同意ではなく実質的な参加が保障されているかが問われます。

## 第6章 日本の法制度との関係

### 6.1 障害者基本法（改正）

2011年の改正で、障害の定義に「社会的障壁」の概念が導入されました。条約の社会モデルを反映した改正です。

### 6.2 障害者差別解消法（2013年制定、2024年改正施行）

- 不当な差別的取扱いの禁止
- **合理的配慮の提供義務**（2024年4月から民間事業者にも法的義務化）
- 就労支援事業所もこの法律の対象

### 6.3 障害者雇用促進法

- **法定雇用率**: 2024年4月から2.5% → **2026年7月から2.7%に引き上げ**
- 対象企業: 従業員37.5人以上（2026年7月〜）
- 週10時間以上20時間未満の短時間労働者も算定対象に（2024年〜）
- 合理的配慮の提供義務
- 障害者雇用に関する優良な中小事業主に対する「もにす認定制度」

### 6.4 障害者総合支援法 — 就労系福祉サービス

| サービス | 概要 | 対象 |
|---------|------|------|
| 就労選択支援 | 就労先・働き方の選択を支援（2025年10月〜） | 就労を希望する障害者 |
| 就労移行支援 | 一般就労に向けた訓練・支援（原則2年） | 一般就労を希望する65歳未満の障害者 |
| 就労継続支援A型 | 雇用契約に基づく就労の機会提供 | 一般企業への就職が困難な障害者 |
| 就労継続支援B型 | 雇用契約を結ばない就労の機会提供 | A型・一般就労が困難な障害者 |
| 就労定着支援 | 就職後の職場定着を支援（最大3年） | 一般就労に移行した障害者 |

### 6.5 B型事業所の工賃の現実

令和5年度（2023年度）の就労継続支援B型の全国平均工賃月額は **17,031円**（※従来の算定方法）です。

条約が求める「同一価値の労働についての同一報酬」（第27条(b)）と、月額約1.7万円の工賃の間には、極めて大きな乖離があります。この構造的問題をどう理解し、どう変えていくかが問われています。

## 第7章 現場で使える条約の視点チェックリスト

### 日常業務の点検ポイント

- [ ] 個別支援計画の策定に利用者本人が実質的に参加しているか
- [ ] 「本人の意思」は形式的な署名ではなく、十分な情報提供と熟慮の上での選択か
- [ ] B型にいる利用者に、A型や一般就労への移行の可能性を定期的に検討しているか
- [ ] 利用者の「できないこと」ではなく、「社会的障壁」に着目しているか
- [ ] 合理的配慮の提供を記録し、その効果を検証しているか
- [ ] 障害のある女性や重度障害者への複合的差別に配慮しているか
- [ ] 作業内容と工賃の関係を利用者に説明し、透明性を確保しているか
- [ ] 苦情申立の仕組みが利用者にとってアクセシブルか

### 意思決定支援の実践ポイント

1. **情報提供**: 選択肢を分かりやすく提示する（絵カード、やさしい日本語など）
2. **時間の確保**: 決定を急がせない
3. **環境調整**: 意思表示しやすい場所・タイミングを選ぶ
4. **最善の解釈**: 本人の表現を「できない」ではなく「こう伝えようとしている」と読み取る
5. **記録**: 意思決定のプロセスと結果を記録する

## 第8章 よくある誤解と正しい理解

### 誤解1: 「B型事業所は条約違反だ」
**正しい理解**: 条約はB型事業所そのものを禁止しているわけではありません。しかし、B型がゴールになってしまい、開かれた労働市場への移行の努力がなされていない状態は問題です。B型は「通過点」または「本人が選択した場」であるべきです。

### 誤解2: 「合理的配慮は予算がないからできない」
**正しい理解**: 合理的配慮は「過度の負担」にならない範囲で提供するものです。多くの合理的配慮はコストをかけずに実現できます（休憩時間の柔軟な設定、作業手順の視覚化、静かな作業スペースの確保など）。

### 誤解3: 「重度の知的障害者には自己決定は無理」
**正しい理解**: 条約は意思決定の「能力」ではなく「機会」を保障しています。表情、行動、好み等から本人の意思を読み取る努力が求められます。「決められない」のではなく「決める支援が足りない」と考えます。

### 誤解4: 「障害者雇用率を満たしていれば条約を守っている」
**正しい理解**: 雇用率は量的な指標に過ぎません。条約は雇用の質（合理的配慮、公正な労働条件、昇進の機会、同一価値同一報酬）も求めています。

## 用語集

| 用語 | 英語 | 説明 |
|------|------|------|
| CRPD | Convention on the Rights of Persons with Disabilities | 障害者権利条約 |
| 社会モデル | Social Model of Disability | 障害を社会的障壁の問題として捉える考え方 |
| 合理的配慮 | Reasonable Accommodation | 障害者が他の者と平等に権利を享有するための必要かつ適当な変更・調整 |
| インクルージョン | Inclusion | 排除せず包容すること |
| シェルタード・ワークショップ | Sheltered Workshop | 保護された就労環境（日本のB型事業所等に相当） |
| 法的能力 | Legal Capacity | 権利の主体として法的行為を行う能力 |
| 支援付き意思決定 | Supported Decision-Making | 本人の意思決定を他者が支援する仕組み |
| 代行決定 | Substituted Decision-Making | 本人に代わって他者が決定する仕組み（条約は転換を求めている） |
| 脱施設化 | Deinstitutionalization | 施設から地域生活への移行 |
| 総括所見 | Concluding Observations | 条約委員会が各国に出す審査結果と勧告 |
| 建設的対話 | Constructive Dialogue | 条約委員会と締約国政府との審査対話 |
| 一般的意見 | General Comment | 条約委員会が特定の条文について出す解釈指針 |
