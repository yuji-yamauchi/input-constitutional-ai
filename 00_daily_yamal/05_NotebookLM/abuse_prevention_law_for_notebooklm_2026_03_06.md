## 第1章：障害者虐待防止法とは何か

### 1.1 法律の基本情報

- **正式名称**: 障害者虐待の防止、障害者の養護者に対する支援等に関する法律
- **制定**: 2011年6月（平成23年）
- **施行**: 2012年10月1日
- **背景**: 障害者権利条約（CRPD）批准に向けた国内法整備の一環として制定
- **同年の重要法改正**: 障害者基本法改正（「社会的障壁」概念の導入）

### 1.2 法律の目的

障害者に対する虐待の禁止、虐待の予防および早期発見、虐待を受けた障害者の保護と自立支援、養護者に対する支援を行うことで、障害者の権利利益の擁護に資することを目的とする。

### 1.3 3つの虐待類型（場面別）

| 類型 | 対象 | 通報先 |
|------|------|--------|
| **養護者による虐待** | 家族・同居人等 | 市町村 |
| **障害者福祉施設従事者等による虐待** | 施設・事業所の職員 | 市町村または都道府県 |
| **使用者による虐待** | 雇用主・事業主等 | 市町村または都道府県 |

※ 2022年の精神保健福祉法改正（2024年4月施行）により、**精神科病院における虐待**も通報制度が義務化された。

### 1.4 5つの虐待行為

| 虐待行為 | 具体例 |
|----------|--------|
| **身体的虐待** | 殴る、蹴る、縛り付ける、閉じ込める、不要な身体拘束 |
| **心理的虐待** | 怒鳴る、無視する、侮辱的な言動、威圧的な態度 |
| **性的虐待** | 性的な行為の強要、わいせつな言葉 |
| **放棄・放置（ネグレクト）** | 食事・入浴・排泄等の介護放棄、必要な医療を受けさせない |
| **経済的虐待** | 年金・賃金の搾取、財産の不正使用、必要な金銭を使わせない |

### 1.5 通報義務

障害者虐待防止法の最大の特徴は、**すべての国民に通報義務がある**こと。

- 虐待を受けたと思われる障害者を発見した者は、速やかに市町村に通報しなければならない（第7条）
- 障害者福祉施設従事者等は、虐待を受けたと思われる場合の通報義務がさらに明確（第16条）
- 通報者は、通報したことを理由とする不利益な取扱いを受けない（通報者保護）
- 通報は「虐待を受けたと思われる」段階で行う ― **確証は不要**

## 第2章：最新の虐待統計（令和6年度調査結果）

| 相談・通報件数 | **11,656件** | +1,684件（+16.9%） |

| 被虐待者数 | **2,518人** | +233人（+10.2%） |

**虐待行為の類型**:

- 精神障害: **47.6%** ← 初めて最多に
- 知的障害: 43.0%

**発生要因**:
- 家庭における人間関係: 45.0%
- **虐待者が虐待と認識していない: 40.1%** ← 重要

**虐待を行った職種**:

### 2.4 統計データが示す構造的問題

1. **「虐待と認識していない」が4割**: 善意で支援しているつもりでも虐待になりうるという構造的問題
2. **発生要因の上位3つがすべて「システムの問題」**: 個人の悪意ではなく、教育不足・理念欠如・ストレスが原因
3. **精神障害が被虐待者で初めて最多に**: 「見えにくい障害」への理解不足が顕在化
4. **警察経由の通報が半数超**: 福祉の内側からの発見が不十分であることの裏返し

## 第3章：法制度の歴史的位置づけ

### 3.1 虐待防止法の前史

| 年 | 出来事 | 意義 |
|----|--------|------|
| 2000年 | 児童虐待防止法制定 | 子どもへの虐待防止が初めて法制化 |
| 2001年 | DV防止法（配偶者暴力防止法）制定 | 家庭内暴力への法的介入 |
| 2006年 | 高齢者虐待防止法施行 | 高齢者への虐待防止 |
| **2011年** | **障害者虐待防止法制定** | **障害者の虐待防止が最後に法制化** |
| 2012年 | 障害者虐待防止法施行 | |
| 2024年 | 精神保健福祉法改正施行 | 精神科病院での虐待通報義務化 |

**注目すべき点**: 児童（2000年）→ 配偶者（2001年）→ 高齢者（2006年）→ **障害者（2011年）**。障害者の虐待防止法制化は最も遅かった。

### 3.2 CRPD批准と虐待防止法の関係

障害者虐待防止法は、日本がCRPDを批准するための国内法整備ラッシュの中で生まれた。

**CRPD第16条（搾取、暴力及び虐待からの自由）**:
> 締約国は、家庭の内外におけるあらゆる形態の搾取、暴力及び虐待（性別に基づくものを含む）から障害者を保護するためのすべての適当な立法上、行政上、社会上、教育上その他の措置をとる。

**2022年 CRPD対日総括所見の関連指摘**:
- 93項目の懸念、92項目の勧告
- 虐待防止に関しても、通報後の対応体制の不十分さ、施設内での権利侵害の継続が指摘された

### 3.3 報酬改定による制度的対応（2024年度〜）

- **虐待防止措置未実施の場合の減算措置**: 虐待防止委員会の未開催、虐待防止責任者の未配置等に対する報酬減算
- **身体拘束廃止未実施減算の引上げ**: 身体拘束の適正化が不十分な場合の減算強化
- **第三者評価の定期的導入の義務化**: 外部の目による監視の制度化

## 第4章：支援現場で知っておくべきこと

### 4.1 通報義務の実際

支援者にとって最も身近な場面は、虐待の疑いに気づいた時の対応である。

**通報のハードル（よくある躊躇の理由）**:
- 「確証がないから通報できない」→ **確証は不要。「思われる」段階で通報義務が発生する**
- 「通報したら利用者さんとの関係が壊れる」→ **通報者情報は守られる**
- 「上司に相談してから」→ **組織内の相談と市町村への通報は別。通報は個人の義務**

**通報しなかった場合のリスク**:
- 虐待のエスカレート
- 他の利用者への波及
- 事業所の管理体制の問題として行政指導の対象に

### 4.2 「虐待と認識していない」問題

統計データで**40.1%**の虐待者が「虐待と認識していない」。これは支援の現場で最も危険な状態。

**グレーゾーンの例**:
- 「この子のためだから」と称する過度な行動制限
- 「安全のため」と称する不必要な身体拘束
- 「指導」と称する威圧的な態度・大きな声
- 本人の意思を確認せずに行うルーティン化した支援
- 「みんなそうしている」という慣習への無批判な追従

### 4.3 身体拘束の適正化

身体拘束は原則禁止。例外的に認められるのは以下の3要件をすべて満たす場合のみ。

| 要件 | 内容 |
|------|------|
| **切迫性** | 本人または他者の生命・身体に危険が差し迫っている |
| **非代替性** | 身体拘束以外に方法がない |
| **一時性** | 一時的なものである |

※ 3要件を満たす場合でも、記録の作成・家族への説明・定期的な見直しが必須。

### 4.4 虐待防止のための組織体制（義務化事項）

2024年度報酬改定により、すべての障害福祉サービス事業所に以下が義務化された。

1. **虐待防止委員会の設置・定期開催**
2. **虐待防止責任者の配置**
3. **従業者への虐待防止研修の実施**（年1回以上）
4. **虐待防止のための指針の整備**

未実施の場合、報酬減算の対象となる。

## 第5章：虐待はなぜ起きるのか ― 構造的理解

### 5.1 善意は虐待を防げない

施設従事者による虐待の発生要因を改めて見ると、上位3つはいずれも個人の悪意ではなく**システムの問題**である。

| 要因 | 割合 | 本質 |
|------|------|------|
| 教育・知識・介護技術等に関する問題 | 67.5% | 研修不足・学ぶ機会がない |
| 倫理観や理念の欠如 | 60.2% | 「なぜこの仕事をしているか」の共有不足 |
| 職員のストレス・感情コントロールの問題 | 58.7% | 労働環境の問題 |

**つまり**: 善意のある人でも、適切な教育がなく、理念の共有がなく、ストレスフルな環境に置かれれば、虐待は起きうる。虐待防止は「悪い人を排除する」ことではなく、**構造を変える**ことである。

### 5.2 虐待の感染モデル（Abuse Infection Model）

虐待には「感染」的な側面がある。

**感染サイクル**:
1. **被害を受ける** → 自尊感情の低下（「自分には価値がない」）
2. **より弱い者への加害** → 「自分にも力がある」という体験
3. **加害の「成功」** → 責任を問われない、検挙されない
4. **反復** → 「自分は特別だ」（傲慢因子の醸成）

このサイクルには2つの因子が関与する。

- **アノミー因子（Anomie Factor）**: 「持たざる者」の規範喪失 ― ルールを守る意味を見出せなくなる（下からの腐敗）
- **ヒュブリス因子（Hubris Factor）**: 「持つ者」の規範喪失 ― ルールを超えていると感じる（上からの腐敗）

**重要**: 虐待の感染モデルでは、最初は「持たざる者」だった人も、加害の成功体験を通じてヒュブリス因子を獲得しうる。

### 5.3 世代間伝達

虐待の影響は当事者だけにとどまらず、次の世代に「相続」される。

- 家庭内で虐待を受けた子どもは、ACEs（小児期逆境体験）として長期的な影響を受ける
- ACEsの数が多いほど、成人後の身体的・精神的健康、就労、社会参加に悪影響
- CRC（子どもの権利条約）第19条は「虐待からの保護」を締約国の義務と規定

### 5.4 「個人の責任」から「社会の債務」へ

虐待事件が起きたとき、社会は「あの職員が悪い」と個人を責める。しかし構造的に見ると、その背景には社会が果たすべきだった義務の未履行がある。

- 十分な研修を提供しなかった組織の責任
- 適切な人員配置をしなかった制度の責任
- ストレスフルな労働環境を放置した社会の責任

虐待を「個人の問題」とする見方は、医学モデルで障害を「個人の問題」とした見方と同じ構造である。虐待防止にも**社会モデル的な視点**が必要である。

## 第6章：支援者としてのセルフチェック

### 6.1 日常のふりかえりチェックリスト

以下の問いを定期的に自分に問いかけてみてください。

- [ ] 利用者さんの意思を確認せずに、こちらの都合で支援内容を決めていないか
- [ ] 「この人にはこれが一番いい」と、本人の声を聞かずに判断していないか
- [ ] イライラした時に、声のトーンや態度が変わっていないか
- [ ] 「みんなそうしている」という理由だけで、疑問を感じる支援を続けていないか
- [ ] 利用者さんの金銭管理に関して、透明性が確保されているか
- [ ] 身体拘束について、3要件（切迫性・非代替性・一時性）を毎回確認しているか
- [ ] 同僚の支援で気になることがあった時、それを伝える手段があるか

### 6.2 困った時の相談先

| 相談先 | 電話番号 | 対象 |
|--------|----------|------|
| 市町村障害者虐待防止センター | 各市町村窓口 | 養護者・施設従事者による虐待 |
| 都道府県障害者権利擁護センター | 各都道府県窓口 | 使用者による虐待、広域相談 |
| 障害者虐待防止ホットライン（仮称） | ※ 各自治体で異なる | 全般的な相談 |

## 第7章：まとめ ― 虐待防止は「プロの仕事」

### 7.1 この動画で伝えたい3つのこと

1. **虐待は「悪い人」だけがするのではない**: 教育不足・理念欠如・ストレスという構造的要因が4割の「自覚なき虐待」を生んでいる
2. **通報は義務であり、守られる**: 「思われる」段階で通報する。確証は不要。通報者は保護される
3. **虐待防止は構造の問題**: 個人の善意だけでは防げない。仕組み（研修・チェック体制・外部評価）で防ぐ

### 7.2 あなたへの問い

> 「あなたの職場で、虐待の芽に気づける仕組みはありますか？」

善意は出発点にすぎない。虐待を防ぐのは、知識と仕組みと、「おかしいと思ったら声を上げる」文化である。それがプロの支援者としてのスタートラインである。
