# 09_research_proposal  
**研究計画書プロジェクト（修士・博士前期）統合ディレクトリ**

このディレクトリは、Puu（福祉パンカー）が大学院へ提出する  
**研究計画書（Research Proposal）を体系的に構築するための中枢フォルダ**です。

計画書の骨格設計・本文作成・参考文献管理・倫理検討・研究プロセス管理を  
フェーズごとに分離し、AI（Claude / ChatGPT）や GitHub 連携による作業効率化を最大化する  
“研究開発環境” として設計されています。

---

## 📁 フォルダ構造と役割

### ### 00_README.md  
**プロジェクト全体のガイドライン（この文書）**  
- ディレクトリの目的  
- 各フォルダの役割  
- 今後の作業方針やTODOを書くトップレベル文書  
- 指導教員や外部第三者が見ても理解できるように整理する場所  

---

### 📁 01_planning_sessions  
**思考ログ・議論記録（YAML / Markdown）置き場**

- Claude / ChatGPT / 実在の研究者との議論ログ  
- YAML形式のメタ構造化したセッション記録  
- アイデア抽出・哲学的検討・火文化的メタ思索などの“素材”  
- **ここは本番では使わないが、必ず計画書の質を底上げする重要なアーカイブ**

👉 *研究計画書の「知的下地」をここで蓄積する。*

---

### 📁 02_outline  
**研究計画書の骨格（アウトライン）**

- 研究背景（Why）  
- 問題設定（What）  
- 目的・仮説（Purpose）  
- 方法論（How）  
- 新規性・独自性（Significance）  
- スケジュール（Timeline）

提出物の前段階として、  
**まず構造を作り、それを03_main_proposalで文章化するための設計図**。

👉 *ここが整えば、研究計画は“ほぼ完成”と言える。*

---

### 📁 03_main_proposal  
**最終的に提出する研究計画書（本文）**

- 3000字〜4000字の和文研究計画書  
- 英文版（任意：海外向け）  
- バージョン管理された草稿・修正稿  
- PDFエクスポート用テンプレート  

👉 *ここが入試本番で提出される “成果物フォルダ”。*

---

### 📁 04_references  
**参考文献フォルダ（研究の根拠と理論背景）**

- ケイパビリティ・アプローチ（Sen / Robeyns）  
- ICF（環境因子）  
- 障害福祉・就労支援の先行研究  
- 経済哲学・社会制度論（資本主義・民主主義・自己責任論）  
- Puu独自のSoEメタファーの哲学的基盤  
- APA形式の文献リスト（bibtex も可）  

👉 *学術的正当性（legitimacy）を担保する重要ファイル群。*

---

### 📁 05_ethics_dialogue  
**研究倫理に関する対話とドキュメント**

- STROBE / CRPD / 福祉研究の倫理枠組への適合性  
- AI活用倫理の検討  
- 当事者性・データ保護・心理的安全性の論点整理  
- Claude や ChatGPT とのメタ倫理対話ログ  

👉 *研究計画書の「倫理的配慮」セクションに直接反映される素材置き場。*

---

### 📁 06_research_process  
**研究プロセス・ロードマップ管理**

- 修士課程2年間のスケジュール  
- 月次計画（Milestones）  
- 文献レビュー計画  
- 調査・データ収集計画（必要に応じて）  
- 実際に提出した修正履歴や進捗ログ  

👉 *研究プロジェクトを“走らせる”ための進行管理フォルダ。*

---

# 📌 ワークフロー（推奨）

01_planning_sessions

# 🧠 Claude / ChatGPT の使い分け（推奨）

- **Claude Code（VS Code）**  
  → YAML統合、構造チェック、論理矛盾の検出、フォルダ横断

- **ChatGPT（構文化・文章化）**  
  → ナラティブ整形、日本語→英語の自然化、読みやすさ調整

- **Claude（ブラウザ）**  
  → CA/ICF・哲学領域の深掘りや批判的検討

---

# ✨ このプロジェクトのゴール

- 日本の障害福祉支援における **本人主権型アセスメントモデル（SoE）** を  
  学術的に正当化し、国内外の大学院に提出できる研究計画書として完成させること。

- 社会制度 × ケイパビリティ × ICF × 会計メタファー を統合した  
  **新しい障害福祉研究の基盤モデルを構築すること。**
