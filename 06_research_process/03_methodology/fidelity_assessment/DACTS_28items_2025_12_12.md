# DACTS（Dartmouth Assertive Community Treatment Scale）28項目詳細

**作成日**: 2025-12-12
**出典**: Teague GB, Bond GR, Drake RE (1998). Program fidelity in assertive community treatment: Development and use of a measure. *American Journal of Orthopsychiatry* 68: 216-232.
**GPT調査報告書に基づく整理**

---

## 概要

Dartmouth Assertive Community Treatment Scale (DACTS)は、包括型地域生活支援（ACT）のプログラムがモデルにどの程度忠実に実施されているかを評価するための尺度である。全**28項目**からなり、**人材・構造**、**組織的境界**、**サービス内容**の3つのカテゴリに分類されている。

### 評価基準
各項目は**5段階**で評定され：
- **1**: 未実施
- **5**: 完全実施

---

## カテゴリ1: 人材・構造（Human Resources: Structure & Composition）

### H1. Small Caseload（少人数担当制）

| 項目 | 内容 |
|------|------|
| **測定目的** | ACTチームのスタッフ一人当たりの利用者数を小さく保ち、サービスの**密度と個別性**を確保すること。適切な担当者数比により、利用者一人ひとりに十分な時間と支援を提供できることを目的とする |
| **定義** | 利用者：スタッフ比が**10:1程度**の小規模な担当制を維持すること。臨床スタッフ1名あたりおおよそ10名以内の利用者を担当 |
| **測定方法** | チームリーダーへのインタビューやプログラムの記録から、チームの常勤換算スタッフ数とアクティブな利用者数を把握し、その比率を計算。直接サービス提供に従事する全スタッフ（精神科医や事務スタッフを除く）を数え、利用者総数との比率を算出。この比率が約10:1以下であれば高忠実度と評価 |
| **分類** | **定量的**（スタッフ対利用者の数値比率） |

---

### H2. Team Approach（チームアプローチ）

| 項目 | 内容 |
|------|------|
| **測定目的** | チーム全体で利用者を支える**チームアプローチの徹底度**を評価。個々の職員ではなくチーム全員が利用者のケアに関与し、責任を共有しているかを見る |
| **定義** | 提供者集団が個々の専門職としてではなく**チームとして機能**し、スタッフ全員がすべての利用者を把握して関わっていること |
| **測定方法** | 一定期間（例：直近2週間）の記録を調べ、**複数のスタッフが各利用者と対面接触している割合**を算出。2週間で90%以上の利用者が複数のスタッフと面接していればチームアプローチが十分実践されている（評価5）とみなされる |
| **分類** | **定量的**（複数スタッフ関与率） |

---

### H3. Program Meeting（プログラムミーティング）

| 項目 | 内容 |
|------|------|
| **測定目的** | チームが利用者支援の計画・レビューのために**定期的かつ頻繁に会議**を行っているかを評価。頻繁な会議はチーム内で情報共有し質の高いケアを提供する目的がある |
| **定義** | チームが**頻回にミーティング**を開き、すべての利用者についてサービス計画を立て振り返っていること。理想的には平日毎日（週4〜5回）は朝会等で全利用者の状況を簡単にでも確認する状態 |
| **測定方法** | チームリーダーやスタッフへの聞き取り、およびスケジュールの確認によって会議開催頻度を把握。週4回以上会議を開き各利用者を毎回検討している場合に最高評価 |
| **分類** | **定性的**（頻度を段階評価、一部定量的要素あり） |

---

### H4. Practicing Team Leader（チームリーダーも実践を行う）

| 項目 | 内容 |
|------|------|
| **測定目的** | チームリーダー（管理・監督者）が**直接サービス提供にどれだけ関与**しているかを評価。リーダー自ら利用者に関わることでチームの臨床介入の質と一貫性を高める |
| **定義** | 前線スタッフのスーパーバイザーであるチームリーダーが、自ら**直接利用者へのサービス提供**を行っていること。理想はリーダーがその役割の中で**半分以上の時間**を利用者支援にあてている状態 |
| **測定方法** | チームリーダーやスタッフへのインタビューで、リーダーの業務内容や時間配分を確認。リーダーの時間の○%を直接支援に充てているかで評価。**50%以上**直接サービス提供していれば評価5、25%未満なら評価が低くなる |
| **分類** | **定量的**（リーダー業務時間中の直接支援時間の割合） |

---

### H5. Continuity of Staffing（スタッフの継続性）

| 項目 | 内容 |
|------|------|
| **測定目的** | スタッフの**定着率**を評価し、頻繁な人員入替がないかを見る。スタッフ継続はチームの凝集性と利用者との継続関係に寄与する |
| **定義** | プログラムが**同じスタッフを長期に維持**していること。年間または2年間でのスタッフ離職・異動率が低い状態 |
| **測定方法** | 過去1〜2年のスタッフ異動記録やチームリーダーへの聞き取りにより、スタッフの**離職率（ターンオーバー率）**を算出。2年間で離職率が**20%未満**であれば高評価、80%以上であれば低評価 |
| **分類** | **定量的**（離職率の数値） |

---

### H6. Staff Capacity（スタッフの欠員がないこと）

| 項目 | 内容 |
|------|------|
| **測定目的** | チームのスタッフポジションが常に**定員充足**されているか、欠員が最小限に留まっているかを評価。十分なスタッフを確保しサービスの継続性を維持する |
| **定義** | プログラムが**スタッフ欠員なしで運営**されていること。必要定数に対し常にほぼ100%充足している状態 |
| **測定方法** | 人員配置表や採用・離職記録を確認し、過去一定期間（例えば1年間）での欠員期間の長さや充足率を評価。年間を通じて**95%以上**の期間定員が満たされていれば最高評価 |
| **分類** | **定量的**（スタッフ充足率） |

---

### H7. Psychiatrist on Staff（精神科医がスタッフにいること）

| 項目 | 内容 |
|------|------|
| **測定目的** | チームに**専任の精神科医**が配置されているか、その十分性を評価。精神科医の関与は医療面のモニタリングや治療計画への参加を確保する |
| **定義** | **100人あたり少なくとも1名の常勤精神科医**がチームに割り当てられていること。フルサイズのチーム（約100名規模）ならフルタイム精神科医1名以上、50名規模なら0.5相当などが目安 |
| **測定方法** | チームのスタッフ名簿から精神科医の在籍と勤務時間（常勤換算FTE）を確認。利用者数と精神科医FTEの比率を算出し、おおむね**100:1の比率**が保たれていれば評価5 |
| **分類** | **定量的**（精神科医FTE対利用者数の比率） |

---

### H8. Nurse on Staff（看護師がスタッフにいること）

| 項目 | 内容 |
|------|------|
| **測定目的** | チームに**専任看護師**が十分配置されているかを評価。看護師は服薬管理や身体ケア面で重要な役割を果たす |
| **定義** | **100人規模のプログラムに対し2名以上の常勤看護師**がいること。チームの規模に応じた看護師数の配置が求められる |
| **測定方法** | スタッフリストを確認し、看護師資格を持つ職員の人数と勤務形態を把握。利用者数100あたり看護師2名相当（**FTE2.0**）が配置されているかを確認。長期不在の看護師はカウントしない |
| **分類** | **定量的**（看護師数の充足度） |

---

### H9. Substance Abuse Specialist on Staff（物質依存専門家がスタッフにいること）

| 項目 | 内容 |
|------|------|
| **測定目的** | チームに**物質使用障害の専門スタッフ**（いわゆるDual Diagnosis対応スタッフ）が配置されているか評価。併存する物質依存への適切な介入提供が目的 |
| **定義** | 100人規模のプログラムにつき、**物質乱用治療の訓練または臨床経験が1年以上あるスタッフが2名以上**在籍していること |
| **測定方法** | スタッフの職歴・資格を確認し、該当する物質依存分野のスペシャリストの人数を数える。100人あたり2名以上配置なら評価5、1名のみならやや不十分、0名なら最低評価 |
| **分類** | **定量的**（専門スタッフ数） |

---

### H10. Vocational Specialist on Staff（職業専門家がスタッフにいること）

| 項目 | 内容 |
|------|------|
| **測定目的** | チームに**就労支援の専門スタッフ**が配置されているかを評価。利用者の就労リハビリや職探し支援を重視し、社会復帰を促進する |
| **定義** | **職業リハビリテーションや就労支援の訓練・経験が1年以上あるスタッフが少なくとも1名**、チームに所属していること。可能であれば2名以上が望ましい |
| **測定方法** | スタッフの経歴を確認し、就労支援の専門知識・経験を持つスタッフ数を数える。1名いれば基準クリア（中程度評価）、2名以上配置であれば理想的（高評価） |
| **分類** | **定量的**（専門スタッフ数） |

---

### H11. Program Size（プログラムのサイズ）

| 項目 | 内容 |
|------|------|
| **測定目的** | チームの規模が**十分なスタッフ数**を備え、多様な職種構成とサービス提供の継続性を確保できているかを評価。適切なチーム規模により、包括的なサービス提供体制を維持する |
| **定義** | プログラムの規模が充分で、必要な職種の多様性とカバー範囲を一貫して提供できていること。フルサイズのACTチーム（都会型）なら約**10〜12名のスタッフ編成**が基準 |
| **測定方法** | スタッフ総数（常勤換算）と専門職の内訳を確認し、それがモデル基準を満たすか評価。利用者100名規模なら臨床スタッフ10名以上いれば評価5、7〜9名なら減点。カバーできるサービス範囲（職種の網羅性）も勘案 |
| **分類** | **定量的＋定性的**（スタッフ数および職種バランス） |

---

## カテゴリ2: 組織的境界（Organizational Boundaries）

### O1. Explicit Admission Criteria（明確なエントリー基準）

| 項目 | 内容 |
|------|------|
| **測定目的** | プログラムがサービス対象とする利用者層を**明確に定義**し、適切な入隊基準を運用しているかを評価。対象者の選定が戦略的に行われているかを確認する |
| **定義** | プログラムには、特定の対象者層にサービスを提供するという**明確な使命**があり、**客観的で操作的に定義された入隊基準**を用いて不適切な紹介をふるい落としていること。例として「過去12ヶ月で2回以上の入院歴がある重度精神障害者」など具体的基準を設定 |
| **測定方法** | プログラムの入院（エントリー）ポリシーや資料を確認し、書面で選考基準が明示されているかを見る。スタッフへの聞き取りで、紹介を受ける際にその基準に沿って受け入れ可否判断をしているかを確認。基準が文書化され周知されていれば高評価 |
| **分類** | **定性的**（基準の明確さと運用状況） |

---

### O2. Intake Rate（新規エントリー率）

| 項目 | 内容 |
|------|------|
| **測定目的** | チームが新規利用者の受け入れを**低いペース**で行っているかを評価。急激な受け入れ増によるサービス質低下を防ぎ、安定したサービス環境を維持する |
| **定義** | プログラムは安定したサービス提供のために、**新規利用者の受け入れペースを抑えている**こと。例えば一ヶ月あたりの新規受け入れ人数が少数に限定されている状態 |
| **測定方法** | 過去数ヶ月〜1年の新規受け入れ件数を記録から集計し、その平均ペースを算出。評価基準は例えば「月あたり**4名以下**のペースなら評価5、8名以上なら評価1」など段階づけ |
| **分類** | **定量的**（加入率の数値） |

---

### O3. Full Responsibility for Treatment Services（治療サービスに対する完全な責任）

| 項目 | 内容 |
|------|------|
| **測定目的** | ACTチームが主要な治療・リハビリテーションサービスを**他部署に頼らず自チーム内で提供**しているかを評価。一貫した包括サービス提供体制を維持する |
| **定義** | プログラムがケースマネジメントや精神科サービスに加え、**カウンセリング/心理療法、住宅支援、物質乱用治療、就労支援、リハビリテーション**などのサービスをチーム内で直接提供していること。外部機関に丸投げせず、サービスがチーム内で統合されている状態 |
| **測定方法** | 提供サービスの範囲についてチームリーダーやスタッフに確認し、主要サービス領域を網羅しているかを評価。利用者のサービス利用記録を抽出し、必要なサービスがチーム内スタッフによって提供されている割合を調べる |
| **分類** | **定性的**（一部サービス提供率など定量要素も考慮） |

---

### O4. Responsibility for Crisis Services（救急サービスに対する責任）

| 項目 | 内容 |
|------|------|
| **測定目的** | チームが**24時間体制で緊急時対応の責任**を負っているかを評価。夜間・週末を含め、利用者の精神科危機に即応できる体制整備が目的 |
| **定義** | プログラムが**24時間体制で精神科緊急対応をカバー**していること。利用者が危機に陥った際、常時ACTチームに連絡・対応が可能な状態 |
| **測定方法** | チームの体制資料（オンコール体制表など）やスタッフへの聞き取りで、緊急対応のカバレッジを確認。「**365日24時間いつでもチームに連絡が取れ対応してもらえる仕組み**」があれば最高評価。夜間・休日に他機関へ委託している場合は減点 |
| **分類** | **定性的**（24時間対応体制の有無） |

---

### O5. Responsibility for Hospital Admissions（入院に対する責任）

| 項目 | 内容 |
|------|------|
| **測定目的** | チームが利用者の**精神科入院時にも深く関与**しているかを評価。無計画な入院を減らし、継続的なケアを確保する |
| **定義** | ACTチームが利用者の精神科入院の際に**密接に関与**し、適切な入院判断や手続きに関わっていること。入院が必要な場合、チームが事前評価や搬送に関与する |
| **測定方法** | スタッフや利用者への聞き取り、最近の入院ケースの振り返りから、チームが入院プロセスにどの程度関与したかを確認。「入院判断や手続きをチームが主導・同行したか」「病院との連携があったか」などを評価 |
| **分類** | **定性的**（関与の度合い） |

---

### O6. Responsibility for Hospital Discharge Planning（退院計画に対する責任）

| 項目 | 内容 |
|------|------|
| **測定目的** | チームが利用者の**退院計画に関与**しているかを評価。入院中から関与し退院後の円滑な地域支援につなげる |
| **定義** | プログラムが**精神科病院からの退院計画に関与**していること。入院期間中からチームが利用者と関わり、病院スタッフと協働して退院後の住居や支援を準備する |
| **測定方法** | 最近の退院事例を検討し、ACTチームが退院調整会議への参加や退院後フォローを行っていたか確認。退院前から関与している割合を見る。継続支援が途切れず行われていれば高評価 |
| **分類** | **定性的**（関与の有無と質） |

---

### O7. Time-unlimited Services / Graduation Rate（無期限のサービス／終了率）

| 項目 | 内容 |
|------|------|
| **測定目的** | プログラムが利用者を**任意の期限で卒業・サービス終了にしない方針**を維持しているかを評価。必要な限り継続支援し、短期介入で終わらせないことが目的 |
| **定義** | プログラムには**任意の利用期間制限がなく**、すべての利用者に必要な限り無期限に関わり続けること。「○ヶ月でプログラム終了」といった恣意的な区切りを設けず、利用者が安定して卒業できるまで支援を継続する姿勢 |
| **測定方法** | プログラムポリシーでサービスに時間制限がないことを確認し、過去1年間のサービス終了（卒業）率を計算。任意の理由でプログラムから卒業（計画的終了）した利用者の割合が低いほど忠実度が高い。「過去1年で**95%以上の利用者が継続**してサービスを受けており、任意終了は5%未満」であればモデルに忠実 |
| **分類** | **定量的**（終了率の数値） |

---

## カテゴリ3: サービス内容（Nature of Services）

### S1. Community-based Services（地域ベースのサービス）

| 項目 | 内容 |
|------|------|
| **測定目的** | サービス提供の場所が主に**利用者の生活圏（地域社会）**で行われているかを評価。オフィス内ではなく地域での支援に重点を置くことで、利用者の実生活環境での機能向上を図る |
| **定義** | プログラムが**オフィス外の地域**で利用者の状態モニタリングや生活技能訓練を行っていること。支援の大部分が利用者の自宅や職場、地域社会の場で提供されている |
| **測定方法** | サービス記録をレビューし、対面接触の場所がオフィス内か地域かを集計。全対面支援時間のうち地域で行われた割合が高いほど評価が高く、**80%以上が地域訪問**であれば最高評価 |
| **分類** | **定量的**（地域 vs オフィスでのサービス比率） |

---

### S2. No Dropout Policy（ドロップアウトを出さないポリシー）

| 項目 | 内容 |
|------|------|
| **測定目的** | プログラムが利用者を容易に**サービス中断させず、積極的に関与を維持**する方針かを評価。参加に消極的な利用者でも見放さず支援を継続することが目的 |
| **定義** | プログラムは利用者を**高い割合で継続支援し続ける**（ドロップアウトさせない）こと。利用者がサービスから離脱しないよう粘り強く関与し続ける姿勢 |
| **測定方法** | 過去一定期間に利用者が非計画的にサービス中止（ドロップアウト）した件数を確認し、その割合を算出。年間ドロップアウト率が低いほど良く、「年間利用者のうち自発的離脱が**5%以下**」で評価5。加えて、離脱しそうな際に何度も訪問・連絡するなど努力が見られれば高評価 |
| **分類** | **定量的＋定性的**（ドロップアウト率と対応方針） |

---

### S3. Assertive Engagement Mechanisms（積極的エンゲージメントの仕組み）

| 項目 | 内容 |
|------|------|
| **測定目的** | 利用者のサービス参加を確実にするため、チームが**積極的な関与手段**（アウトリーチや法的手段等）を講じているかを評価。離脱リスクの高い利用者への働きかけを維持する |
| **定義** | プログラムは利用者との継続的なエンゲージメントを確保するために、必要に応じて**ストリートアウトリーチ**（地域や路上への外回り）や**法的手段**（例：保護観察官との連携、入院強制措置の活用）などを用いていること |
| **測定方法** | スタッフへのインタビューで、利用者が連絡を絶ったりサービス参加が難しくなった場合にどのような手段を講じているかを確認。「自宅訪問を増やす」「連絡が取れない場合は保証人や家族に働きかける」「必要に応じて医療保護入院や地域移行支援制度を活用する」等の具体策が取られていれば高評価 |
| **分類** | **定性的**（積極的関与の有無と種類） |

---

### S4. Intensity of Service（サービスの量）

| 項目 | 内容 |
|------|------|
| **測定目的** | 利用者に提供される**サービスの総時間（密度）**が必要に応じて十分確保されているかを評価。重度の症状を持つ利用者の地域生活維持には集中的な支援が必要な場合が多く、その量を測る |
| **定義** | 必要に応じて提供される**対面サービス時間が多い**こと。1人の利用者あたり週数時間以上のフェイス・トゥ・フェイス支援を行っているような状態 |
| **測定方法** | 一定期間（例：1週〜1ヶ月）のサービス記録をサンプル抽出し、各利用者に対する直接サービス提供時間の平均を算出。評価基準は1利用者あたり週あたりのサービス時間で段階づけ。平均で**週2時間未満なら1、2〜3時間で3、4時間以上なら5** |
| **分類** | **定量的**（サービス時間量） |

---

### S5. Frequency of Contact（関わりの頻度）

| 項目 | 内容 |
|------|------|
| **測定目的** | 利用者との**接触回数の頻度**が必要に応じて高い水準にあるかを評価。こまめなコンタクトにより利用者の状況把握と支援を行う |
| **定義** | 必要に応じて提供される**サービス訪問・接触の回数が多い**こと。利用者一人あたり週に複数回の訪問・連絡を行う状態 |
| **測定方法** | サービス記録から、一定期間の対面または電話等での接触回数を各利用者について集計し、その平均あるいは中央値を算出。評価基準は週あたり接触回数で設定され、**週4回以上なら評価5、週1回程度なら評価2** |
| **分類** | **定量的**（接触回数の数値） |

---

### S6. Work with Informal Support System（私的サポートシステムとともに関わる）

| 項目 | 内容 |
|------|------|
| **測定目的** | 利用者の**家族や友人、大家、雇用主など非専門の支援ネットワーク**とも積極的に関わり支援しているかを評価。利用者の生活環境全体を支えるための取り組みを測定する |
| **定義** | 利用者が同席していてもいなくても、チームが利用者の**インフォーマルな支援ネットワーク**（家族、家主、雇用者など）に対して支援や技能訓練を提供していること。プロのスタッフだけでなく周囲の人々を巻き込んで支援する姿勢 |
| **測定方法** | 記録や聞き取りで、家族等への働きかけ頻度を確認。各利用者について家族等に**月1回以上連絡・相談**していれば高評価。チームリーダーに現在支援している利用者のうち家族等と定期的に連絡を取っている人数のリストを用意してもらい、その割合を評価 |
| **分類** | **定量的＋定性的**（ネットワーク関与率と支援内容） |

---

### S7. Individualized Substance Abuse Treatment（物質乱用個人治療）

| 項目 | 内容 |
|------|------|
| **測定目的** | チーム内で**個別の物質乱用治療**（依存症カウンセリング）が提供されているか、その提供率を評価。重複障害（精神障害＋物質使用障害）を持つ利用者への直接的介入を確保する |
| **定義** | 物質使用障害を持つ利用者に対し、チーム内の1名以上のスタッフが**個別の治療（カウンセリング等）を直接提供**していること。該当利用者に対して個別面接形式の依存症支援を行っている状態 |
| **測定方法** | チームの物質使用障害該当者の名簿と治療記録を確認し、何割の該当者が個別SUDカウンセリングを受けているかを算出。該当者のうち**75%以上が週1回以上**スタッフによる依存症カウンセリングを受けていれば高評価。提供頻度も評価に加味 |
| **分類** | **定量的**（個別治療提供率） |

---

### S8. Dual Disorder Treatment Groups（重複診断ケース治療グループ）

| 項目 | 内容 |
|------|------|
| **測定目的** | チームが**精神障害と物質使用障害を併せ持つ人向けのグループ治療**を実施しているかを評価。グループ療法による相乗効果で回復を促す |
| **定義** | プログラムが物質使用障害を持つ人々に対する治療の一環として**グループ形式のアプローチ**を用いていること。重複診断（Dual Diagnosis）の利用者向けのグループセッション（依存症回復グループ等）を開催している状態 |
| **測定方法** | プログラムの活動予定表やスタッフへの聞き取りで、該当グループの開催状況（頻度・参加者数）を確認。「依存症セルフヘルプグループを**週1回開催**し、物質乱用の課題を持つ利用者の半数以上が参加している」場合は高評価。開催が無かったり極めて稀なら低評価 |
| **分類** | **定性的＋定量的**（開催有無と質、頻度・参加率） |

---

### S9. Dual Disorders (DD) Model（重複診断モデル）

| 項目 | 内容 |
|------|------|
| **測定目的** | チームが**重複診断（精神障害＋物質使用障害）に対する段階的治療モデル**を実践しているかを評価。対決しない動機づけ重視のアプローチで統合的治療を行う |
| **定義** | プログラムが**段階的な治療モデル**（ステージごとの介入法）を用いており、利用者を**非対決的**に支援し、行動療法の原理に基づき、精神疾患と物質乱用の相互作用を考慮した形で徐々に断酒・断薬に向けた見通しを持つよう支援していること。モチベーション段階に応じた統合的なDual Diagnosis治療モデル（例えばトランスセオレティカルモデル等）を採用している状態 |
| **測定方法** | 物質使用障害担当スタッフへのインタビューや治療マニュアルの有無を確認し、どのような依存症治療モデルを用いているかを評価。**エビデンスに基づいた段階的アプローチ**（例えば動機づけ面接＋認知行動療法など統合モデル）を実践していれば高評価。伝統的な12ステップや対決的手法のみの場合は低評価 |
| **分類** | **定性的**（治療モデルの適切性） |

---

### S10. Role of Consumers on Treatment Team（治療チームにおけるコンシューマーの役割）

| 項目 | 内容 |
|------|------|
| **測定目的** | 精神障害の**当事者（コンシューマー）をチームのスタッフとして起用**し、サービス提供に直接関与させているかを評価。ピアスタッフの参加によるサービス文化の向上と当事者視点の導入が目的 |
| **定義** | サービス利用者の経験を持つ当事者（**ピアスタッフ**）がチームメンバーとして雇用され、直接サービス提供に関わっていること。ACTチームのスタッフにピアサポーターが含まれており、他のスタッフと同等にケース担当やアウトリーチを行っている状態 |
| **測定方法** | スタッフ名簿や聞き取りで、何名のピアスタッフが在籍し、どの程度サービス提供に携わっているかを確認。**少なくとも1名の常勤ピアスタッフ**がいて積極的に訪問やミーティングに参加していれば高評価。形式的に雇っているだけでサービス提供していない場合は低評価 |
| **分類** | **定性的＋定量的**（役割の実質的な有無、ピアスタッフ人数） |

## DACTSの改訂版・最新版と利用条件

### 改訂版：TMACT（Tool for Measurement of ACT）

DACTSには改訂版として**TMACT（Tool for Measurement of ACT）**がある。

| 項目 | DACTS | TMACT |
|------|-------|-------|
| **開発年** | 1998年 | 2012年 |
| **項目数** | 28項目 | 47項目 |
| **変更点** | - | DACTSから6項目削除、25項目追加 |
| **最新版** | 2017年改訂版 | Version 1.0 Revision 3（2018年） |
| **特徴** | 基本的フィデリティ測定 | より高い基準でACTモデル実践度を評価 |

### 最新版の発行年

- **原版DACTS**: マイナーチェンジが行われており、米国SAMHSAによる2003年版、IUPUI（インディアナ大学）による2011年版を経て、**2017年**にオリジナル開発者の協力のもと編集・改訂された版が報告されている
- **TMACT**: 初版公表は**2012年**、最新版リビジョンの発行は**2018年**

### ライセンス条件（著作権と使用許諾）

- **著作権**: おそらく開発者および発表雑誌（Teagueら1998年の論文）に帰属
- **配布状況**: 米国SAMHSAのACT実施キットに収録されるなど公的資料として広く配布
- **利用料**: **無料**（非商用の目的では自由に使用可能）
- **商用利用**: 原著論文の出版社や著作者への許可申請が望ましい
- **TMACT**: 一般に無償で公開されており、誰でもダウンロードして使用可能。ただし、精度を担保するため**トレーニング受講が推奨**
