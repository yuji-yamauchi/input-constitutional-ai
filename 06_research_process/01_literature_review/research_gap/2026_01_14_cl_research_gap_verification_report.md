### 2. 「実証研究（Empirical Study）」と呼べるものの分析

**重要な定義区分：**

| 研究種別 | 定義 | 国内での該当数（2015-2025） |
|----------|------|------------------------------|
| **尺度開発研究（Scale Development）** | 構成概念の定義、項目作成、因子分析、信頼性・妥当性検証を含む | **0件** |
| **心理測定学的検証（Psychometric Validation）** | Cronbach's α、構造方程式モデリング、予測妥当性検証等 | **0件** |
| **ランダム化比較試験（RCT）** | アセスメント手法の効果検証 | **0件** |
| 実践ツール開発 | 専門家協議と試用評価に基づく | 2件 |
| 実態調査・質問紙調査 | 基礎集計と質的分析 | 3件 |
| 実践マニュアル | 手順書・ガイドライン | 4件 |

### 3. 公式文献における研究不足の自認

**厚生労働省「就労系障害福祉サービスにおける職業的アセスメントハンドブック」（2021）より引用：**

> 「職業評価法が導入されてきたのですが、**職業的アセスメントに関しては、特に近年における実践的な取り組みや研究が不足しています**」（p.29）

この記述は、日本政府機関自身が研究の不足を認めている直接的証拠である。

### 4. JEED報告書No.168（2023）の方法論的検討

JEEDの「就労支援のためのアセスメントシート」開発は以下の方法で行われた：

1. 研究委員会による専門家検討
2. 就労支援機関への質問紙調査
3. 障害者雇用企業への質問紙調査
4. 就労支援機関における試作版の試用評価

**欠如している学術的手続き：**
- 探索的因子分析（EFA）・確認的因子分析（CFA）
- 内的整合性係数（Cronbach's α等）の算出
- 構成概念妥当性の検証
- 基準関連妥当性の検証
- 構造方程式モデリング（SEM）による構造検証

→ これは「実践的ツール開発」であり、「学術的尺度開発研究」ではない。

## Ⅱ. 海外研究の動向

### 1. IPS（Individual Placement and Support）モデルにおける実証研究

| 尺度名 | 開発年 | 心理測定学的特性 | 検証内容 |
|--------|--------|------------------|----------|
| IPS-25 Fidelity Scale | 2012 | α = .88 | 内的整合性、予測妥当性（就労率との相関 r = .34） |
| IPS-Q (Knowledge Quiz) | 2010 | α = .89 | 内容妥当性、弁別妥当性、収束的妥当性 |
| Japanese IPS Fidelity Scale | 2018 | 検証済み | 日本語版の心理測定学的特性検証 |

### 2. ICF Core Set for Vocational Rehabilitation

**国際的開発プロセス（Escorpizo et al., 2011）：**
- 4つの準備研究（系統的レビュー、専門家調査、横断研究、フォーカスグループ）
- 国際コンセンサス会議（23カ国の専門家参加）
- 検証研究の実施

**開発された内容：**
- 包括的コアセット：90カテゴリー（活動・参加40、環境因子33、心身機能17）
- 簡易コアセット：13カテゴリー

### 3. 日本国内における検証研究の不在

ICF Core Set for VRの日本における検証研究は、検索範囲内で確認できなかった。ICF自体は日本で参照されているが、就労リハビリテーション領域における体系的な検証研究は見当たらない。

### 4. RCT（ランダム化比較試験）の件数比較

## Ⅲ. 結論

### 主張の検証結果

**元の主張：**
> 「日本の就労移行支援は制度改革が進む一方、アセスメント構造に関する実証研究はほとんど存在しない」

**検証結果：この主張は妥当である。**

ただし、より精密な記述としては以下を推奨する：

### 推奨される修正表現

**選択肢A（最も正確）：**
> 「日本の就労移行支援において、アセスメント**指標自体の心理測定学的検証**（構成概念妥当性、予測妥当性、信頼性等）に関する学術的実証研究は、検索範囲内でほぼ皆無である。既存の研究は実態調査と実践ガイドライン作成に限定されており、欧米のIPS研究のような尺度開発・検証研究は不在である。」

**選択肢B（簡潔版）：**
> 「日本の就労移行支援領域では、アセスメントツールの構造的妥当性や環境因子の統合に関する学術的実証研究は、欧米と比較して著しく不足している。」

**選択肢C（政府文献に依拠した表現）：**
> 「厚生労働省のハンドブック（2021）においても『職業的アセスメントに関しては、特に近年における実践的な取り組みや研究が不足している』と明記されており、この研究空白は公式に認識されている。」

### 研究計画書への記載推奨

1. **背景セクション**：選択肢AまたはBを使用し、具体的な根拠を付す
2. **先行研究の限界**：JEED報告書No.168と厚労省ハンドブックの方法論的限界を指摘
3. **研究の独創性**：本研究がこのギャップを埋める最初の試みであることを強調
4. **参考文献**：厚労省ハンドブック、JEED報告書、IPS国際研究を引用
