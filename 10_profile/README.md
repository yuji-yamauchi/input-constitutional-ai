# 10_profile - 山内雄司（Puu）プロファイル

研究者・実践者としての経歴、スキル、アセスメント情報を管理するディレクトリ。

## ディレクトリ構造

```
10_profile/
├── README.md                    # このファイル
├── yamauchi_yuji_profile.yaml   # 基本プロファイル
├── career_history.yaml          # 経歴・職歴
├── skills_inventory.yaml        # スキルセット
├── assessments/                 # 各種アセスメント
│   └── 2026_01_22_claude_assessment.yaml
├── public/                      # 公開用プロファイル
│   ├── lancers_profile.md       # ランサーズ用
│   ├── menta_profile.md         # MENTA用
│   └── linkedin_profile.md      # LinkedIn用（将来）
└── references/                  # 推薦状・評価など
```
